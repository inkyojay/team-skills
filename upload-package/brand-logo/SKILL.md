# 브랜드 로고 검색

브랜드 로고를 검색하고 다운로드 링크를 제공합니다.

## 트리거
- "로고 찾아줘"
- "[브랜드명] 로고"
- "로고 다운로드"

## 검색 프로세스

### Step 1: 공식 소스 확인
- 브랜드 공식 웹사이트
- 브랜드 미디어킷/프레스킷
- 공식 SNS

### Step 2: 로고 유형 확인
- 기본 로고 (Full color)
- 흑백 로고 (Monochrome)
- 아이콘/심볼
- 가로형/세로형

### Step 3: 파일 형식
- SVG (벡터, 권장)
- PNG (투명 배경)
- JPG

## 출력 형식

```
# [브랜드명] 로고

## 공식 소스
- 웹사이트: [URL]
- 미디어킷: [URL]

## 로고 파일
| 유형 | 형식 | 링크 |
|------|------|------|
| 기본 | SVG | [다운로드] |
| 흑백 | PNG | [다운로드] |

## 사용 가이드라인
- 최소 크기:
- 여백 규정:
- 금지 사항:
```

## 주의사항
- 저작권 확인 필수
- 공식 소스 우선 사용
- 가이드라인 준수
