# 스킬 생성 가이드

새로운 Claude 스킬을 만드는 방법을 안내합니다.

## 트리거
- "스킬 만들어줘"
- "새 스킬 생성"
- "커스텀 스킬 만들고 싶어"

## 스킬 구조

### 필수 구성요소
```markdown
---
name: skill-name
description: 스킬 설명. 트리거 키워드 포함.
---

# 스킬 제목

## 트리거
- "트리거 문구 1"
- "트리거 문구 2"

## 워크플로우
### Step 1: ...
### Step 2: ...

## 출력 형식
```

### YAML 프론트매터 규칙
- name: 소문자, 숫자, 하이픈만 사용
- description: 트리거 키워드 포함 (자동 감지용)

## 스킬 유형

### 1. 단순 스킬
- 특정 작업 1개 수행
- 명확한 입력/출력

### 2. 에이전트 스킬
- 복합 작업 조율
- 다른 스킬 호출 가능

### 3. 워크플로우 스킬
- 단계별 프로세스
- 사용자 확인 포인트 포함

## 저장 위치
- Claude Code: `~/.claude/commands/`
- Claude.ai: 설정 → 스킬 → 업로드
