# 비디오 스크립트 스킬

마케팅 영상을 위한 스크립트를 작성합니다.

## 영상 유형별 구조

### 1. 인스타그램 릴스 (15-30초)
```markdown
## 구조
[0-3초] 훅: 스크롤 멈추게 하는 첫 장면
[3-20초] 본문: 핵심 메시지
[20-30초] CTA: 행동 유도

## 훅 유형
1. 질문형: "왜 아기가 밤마다 깰까요?"
2. 충격형: "이거 모르면 큰일나요"
3. 결과형: "이렇게 하니까 5시간 잤어요"
4. 공감형: "밤잠 설치는 엄마들 모여"
```

### 2. 유튜브 쇼츠 (30-60초)
```markdown
## 구조
[0-3초] 훅
[3-45초] 본문 (3단계)
[45-60초] CTA + 구독 유도

## 예시
훅: "3일 만에 아기 숙면 성공한 비결"
본문:
1. 문제: "매일 밤 3번씩 깨는 아기"
2. 시도: "여러 방법 다 해봤는데..."
3. 해결: "이 베개로 바꾸고 나서"
CTA: "링크 확인하고 구독!"
```

### 3. 제품 소개 영상 (60-90초)
```markdown
## 구조
[0-10초] 문제 제기
[10-30초] 해결책 제시
[30-60초] 기능/혜택 설명
[60-80초] 사회적 증거
[80-90초] CTA

## 예시 (밤부 베개)
문제: "아기 역류 때문에 걱정되시죠?"
해결: "5° 각도 설계로 해결했어요"
기능: "특허받은 인체공학 + 밤부 소재"
증거: "72,000명 엄마가 선택"
CTA: "지금 바로 확인하세요"
```

### 4. 고객 후기 영상 (30-60초)
```markdown
## 구조
[0-5초] 인트로 (고객 소개)
[5-20초] Before (문제 상황)
[20-40초] After (해결 후)
[40-55초] 추천 이유
[55-60초] CTA

## 인터뷰 질문
1. 구매 전 고민이 뭐였어요?
2. 사용 후 어떻게 달라졌어요?
3. 주변에 추천하고 싶어요?
```

### 5. 언박싱 영상 (60-120초)
```markdown
## 구조
[0-10초] 제품 도착
[10-40초] 박스 오픈
[40-80초] 제품 상세 보기
[80-100초] 첫인상/느낌
[100-120초] 사용 예고

## 포인트
- ASMR 요소 (포장 뜯는 소리)
- 손/제품 클로즈업
- 솔직한 리액션
```

## 스크립트 템플릿

### 릴스 스크립트 (15초)
```
[장면 1: 0-3초]
비주얼: 지친 엄마 표정
나레이션: "밤마다 3번씩 깨는 아기 때문에 지치셨죠?"

[장면 2: 3-8초]
비주얼: 제품 클로즈업
나레이션: "5° 각도 베개 하나로 해결했어요"

[장면 3: 8-12초]
비주얼: 잠자는 아기
나레이션: "이제 5시간 연속으로 자요"

[장면 4: 12-15초]
비주얼: 제품 + 로고
텍스트: "프로필 링크 확인 👆"
```

### 유튜브 쇼츠 스크립트 (45초)
```
[훅: 0-3초]
"아기가 자다가 갑자기 깨서 우는 이유, 알고 계세요?"

[문제: 3-12초]
"모로반사 때문이에요.
갑자기 팔다리가 펴지면서 놀라서 깨는 거죠.
저도 매일 밤 3번씩 깼어요."

[해결: 12-30초]
"그런데 이 슬리핑백 쓰고 나서 달라졌어요.
팔을 감싸주니까 모로반사가 줄어들고,
처음으로 5시간 연속 잤어요!"

[증거: 30-40초]
"저만 그런 게 아니에요.
리뷰 4.9점, 7만 명이 선택했어요."

[CTA: 40-45초]
"아래 링크 확인하고, 구독 꾹!"
```

## 훅 아이디어 (30개)

### 질문형
1. "왜 아기가 밤마다 깰까요?"
2. "역류 방지 베개, 정말 효과 있을까?"
3. "모로반사 언제까지 지속될까요?"

### 공감형
4. "밤잠 설치는 엄마들 여기 모여"
5. "이거 공감되면 좋아요"
6. "아기 재우고 기어나오는 거 나만 그래?"

### 결과형
7. "3일 만에 5시간 숙면 성공"
8. "이거 하나로 밤잠 해결됨"
9. "72,000명이 선택한 이유"

### 비밀형
10. "아무도 안 알려주는 수면 팁"
11. "소아과 의사가 추천하는 이것"
12. "출산 준비 필수템 TOP 3"

### 경고형
13. "이거 모르면 밤새 고생해요"
14. "역류 방지 못 하면 위험해요"
15. "신생아 베개 잘못 고르면..."
