# 메타 광고 이미지 제작 스킬

제품 사진 + 카피를 조합하여 메타 광고용 이미지를 생성합니다.

## 에이전트 연동

이 스킬은 `meta-ad-creator` 에이전트와 연동됩니다. 에이전트는 이 스킬의 템플릿과 스크립트를 활용하여 자동으로 광고를 생성합니다.

- **에이전트 파일**: `agents/meta-ad-creator.md`
- **템플릿 역할**: 아래 11개 템플릿은 **참고용 본판**입니다. 에이전트는 이미지 특성(밝기, 구도, 여백 위치)에 따라 CSS를 자유롭게 변형하여 최적의 광고를 생성합니다.
- **직접 사용**: 에이전트 없이 수동으로 템플릿을 사용할 때는 아래 워크플로우를 따르세요.

## 워크플로우

1. **사용자 입력 확인**: 제품 사진 경로, 제품명, 원하는 스타일
2. **템플릿 선택**: 아래 11가지 중 목적에 맞는 템플릿 선택
3. **HTML 생성**: 템플릿의 플레이스홀더를 실제 값으로 교체하여 `output/광고카피/`에 HTML 저장
4. **PNG 변환**: `generate_ad.py`로 이미지 변환
5. **결과 확인**: 생성된 PNG 이미지 확인 및 필요시 카피/레이아웃 수정

## 템플릿 유형

### A. lifestyle (라이프스타일)
- **용도**: 일상 속 제품 사용 장면, 브랜드 인지도 광고
- **구조**: 제품 사진 전면 배치 + 상단 또는 하단에 카피 오버레이
- **적합**: 감성적 소구, 라이프스타일 컷, UGC 사진 활용
- **템플릿**: `assets/templates/lifestyle.html`

### B. promotion (프로모션/세일)
- **용도**: 할인, 특가, 시즌오프 광고
- **구조**: 상단 이벤트 카피 + 중앙 제품 + 하단 할인가/뱃지
- **적합**: 가격 소구, 기간 한정 프로모션
- **템플릿**: `assets/templates/promotion.html`

### C. event (이벤트/라이브)
- **용도**: 라이브 방송, 이벤트 공지
- **구조**: 상단 브랜드+카피 + 중앙 제품 이미지
- **적합**: 라이브 커머스, 이벤트 안내, 론칭 공지
- **템플릿**: `assets/templates/event.html`

### D. new-product (신제품 출시)
- **용도**: 신제품 런칭, 제품 기능 소개
- **구조**: 상단 60% 제품 사진 + 하단 40% 정보 패널
- **적합**: 신제품 어필, 특징 포인트 강조, 컬러 옵션 소개
- **템플릿**: `assets/templates/new-product.html`

### E. social-proof (리뷰/소셜프루프)
- **용도**: 고객 리뷰 기반 광고, 신뢰도 소구
- **구조**: 제품 사진 배경 + 리뷰 말풍선 + 하단 CTA
- **적합**: 리뷰 마케팅, 사회적 증거 활용
- **템플릿**: `assets/templates/social-proof.html`

### F. us-vs-them (비교 광고)
- **용도**: 자사 제품 vs 경쟁 제품 직접 비교
- **구조**: 좌우 분할 — 왼쪽 경쟁사(회색 처리) + 중앙 VS 뱃지 + 오른쪽 자사(선명) + 하단 스펙 비교표
- **적합**: 차별화 소구, 스펙 우위 강조, 전환 최적화
- **레퍼런스**: The Performers 프레임워크, Huel/Gillette/Bellroy 사례
- **템플릿**: `assets/templates/us-vs-them.html`

### G. audience-callout (타겟 호명)
- **용도**: 특정 타겟 그룹을 직접 호명하여 관련성 극대화
- **구조**: 상단 오렌지 바에 타겟 호명 카피 + 제품 이미지 전면 + 하단 그라데이션 오버레이에 솔루션 카피 + CTA
- **적합**: 타겟 세그먼트별 맞춤 광고, 공감형 훅, CTR 향상
- **레퍼런스**: The Performers 프레임워크, Jones Road Beauty 사례
- **템플릿**: `assets/templates/audience-callout.html`

### H. direct-offer (다이렉트 오퍼)
- **용도**: 할인율/오퍼를 화면의 주인공으로 내세우는 전환 특화 광고
- **구조**: 오렌지 배경 전체 + 상단 대형 할인율 + 중앙 제품 이미지 + 하단 CTA 버튼
- **적합**: 첫 구매 유도, 프로모션 극대화, 리타겟팅 전환
- **레퍼런스**: The Performers 프레임워크, Bonobos/DTC 사례
- **템플릿**: `assets/templates/direct-offer.html`

### I. stress-test (내구성/기능 시연)
- **용도**: 제품의 내구성이나 특정 기능을 디테일컷으로 증명
- **구조**: 상단 제품 디테일 사진(지퍼, 소재, 봉제 등) + 라벨 뱃지 + 하단 기능 카피
- **적합**: 기능 증명, 품질 소구, 디테일 강조
- **레퍼런스**: AdKings 프레임워크, Sheertex 사례
- **템플릿**: `assets/templates/stress-test.html`

### J. ugc-native (UGC 네이티브)
- **용도**: 실제 사용자가 올린 듯한 네이티브 콘텐츠 스타일
- **구조**: 제품 사진(저채도+노이즈 필터) + 인스타 유저 프로필 UI + 하단 그라데이션에 후기 인용 + 해시태그
- **적합**: 광고 피로도 낮춤, 피드 네이티브 도달률 극대화, 메타 알고리즘 선호
- **레퍼런스**: AdKings 프레임워크, Thinx 사례
- **템플릿**: `assets/templates/ugc-native.html`

### K. minimal-product (미니멀 프로덕트)
- **용도**: 넓은 여백과 최소 텍스트로 프리미엄 브랜드 느낌 연출
- **구조**: 아이보리 배경 + 상단 브랜드명 + 중앙 제품 이미지 + 제품명/서브카피 + 하단 CTA
- **적합**: 프리미엄 브랜딩, 텍스트 15% 이하 자동 충족, 브랜드 인지도 광고
- **레퍼런스**: AdKings 프레임워크, Rothy's/YETI/Aarke 사례
- **템플릿**: `assets/templates/minimal-product.html`

## 이미지 변환

```bash
python3 skills/advertising/meta-ad-image/scripts/generate_ad.py \
  --html-file output/광고카피/ad-name.html \
  --output-dir output/광고카피/ \
  --scale 2
```

## 규격 참고
상세 메타 광고 규격은 `references/ad-specs.md` 참조.

## 결과물 저장
모든 결과물은 `output/광고카피/`에 저장합니다.
