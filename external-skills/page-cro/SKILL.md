# 랜딩페이지 CRO 스킬

랜딩페이지의 전환율을 분석하고 개선 방안을 제안합니다.

## 분석 프레임워크

### 1. 첫인상 (Above the Fold)
- [ ] 가치 제안이 명확한가?
- [ ] 3초 안에 무엇을 하는 페이지인지 알 수 있는가?
- [ ] CTA 버튼이 눈에 띄는가?
- [ ] 히어로 이미지가 제품/서비스를 잘 보여주는가?

### 2. 가치 제안
- [ ] 고객의 문제를 명확히 언급하는가?
- [ ] 해결책이 구체적인가?
- [ ] 경쟁사 대비 차별점이 있는가?
- [ ] 숫자/데이터로 증명하는가?

### 3. 신뢰 요소
- [ ] 고객 후기/리뷰가 있는가?
- [ ] 미디어 노출/인증 마크가 있는가?
- [ ] 구체적인 사례 연구가 있는가?
- [ ] 보증/환불 정책이 명시되어 있는가?

### 4. CTA (Call-to-Action)
- [ ] CTA 버튼 텍스트가 행동 지향적인가?
- [ ] 버튼 색상이 대비되는가?
- [ ] 페이지 내 CTA가 충분한가? (스크롤 대비)
- [ ] 긴급성/희소성 요소가 있는가?

### 5. 폼 최적화
- [ ] 입력 필드 수가 최소화되어 있는가?
- [ ] 각 필드의 목적이 명확한가?
- [ ] 에러 메시지가 친절한가?
- [ ] 진행 상태가 표시되는가?

### 6. 모바일 최적화
- [ ] 모바일에서 CTA가 탭하기 쉬운가?
- [ ] 텍스트가 읽기 쉬운 크기인가?
- [ ] 로딩 속도가 빠른가?
- [ ] 스크롤이 자연스러운가?

## 개선 우선순위 매트릭스

| 우선순위 | 영향도 | 난이도 | 예시 |
|---------|--------|--------|------|
| 🔴 P0 | 높음 | 낮음 | CTA 문구 변경, 색상 대비 |
| 🟠 P1 | 높음 | 중간 | 가치 제안 재작성, 신뢰 요소 추가 |
| 🟡 P2 | 중간 | 낮음 | 이미지 최적화, 폼 간소화 |
| 🟢 P3 | 낮음 | 높음 | 전체 리디자인, A/B 테스트 인프라 |

## 출력 형식

```markdown
# [페이지명] CRO 분석 리포트

## 요약
- 전체 점수: X/100
- 주요 개선점: 3가지

## 상세 분석

### ✅ 잘하고 있는 점
1. ...
2. ...

### ⚠️ 개선 필요
1. [P0] ...
2. [P1] ...

### 💡 A/B 테스트 아이디어
1. 헤드라인: "A" vs "B"
2. CTA 색상: 현재 vs 대비색
```

## 썬데이허그 체크리스트 (추가)

- [ ] 아기 이미지가 감성적인가?
- [ ] 부모의 고민 (수면, 안전)을 공감하는가?
- [ ] 프리미엄 느낌이 전달되는가?
- [ ] 밤부 소재 등 차별점이 강조되는가?
