# 콘텐츠 검토

생성된 콘텐츠의 품질을 검토하고 개선점을 제안합니다.

## 트리거
- "콘텐츠 검토해줘"
- "품질 체크"
- "이거 괜찮은지 확인해줘"

## 검토 기준

### 1. 브랜드 일관성
- 톤앤매너 적합성
- 브랜드 가이드라인 준수
- 금지 표현 사용 여부

### 2. 콘텐츠 품질
- 문법 및 맞춤법
- 가독성
- 논리적 흐름

### 3. 마케팅 효과성
- 타겟 고객 적합성
- CTA 명확성
- 차별화 포인트

### 4. 기술적 완성도
- HTML 유효성
- 이미지 최적화
- 모바일 대응

## 출력 형식

```
# 콘텐츠 품질 리포트

## 총점: X/10

## 체크리스트
✅ 브랜드 톤앤매너 적합
✅ 문법/맞춤법 OK
⚠️ CTA 강화 필요
❌ 이미지 alt 누락

## 개선 제안
1. [구체적 수정 사항]
2. [구체적 수정 사항]

## 수정된 버전
[개선된 콘텐츠]
```
