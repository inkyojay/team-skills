# 경쟁사 분석

경쟁 브랜드를 분석하고 전략적 대응 방안을 제시합니다.

## 트리거
- "경쟁사 분석해줘"
- "[브랜드명] 분석"
- "벤치마킹 리포트"

## 분석 프로세스

### Step 1: 정보 수집
- 공식 홈페이지, 스마트스토어, SNS 검색
- 제품 라인업 및 가격 수집
- 베스트셀러 제품 파악

### Step 2: 마케팅 분석
- SNS 팔로워, 게시물 빈도, 인게이지먼트
- 광고 활동 및 인플루언서 협업 현황

### Step 3: 고객 반응 수집
- 리뷰 분석 (긍정/부정 키워드)
- 맘카페, 육아 커뮤니티 반응

### Step 4: 리포트 작성

## 출력 형식

```
# [브랜드명] 경쟁사 분석 리포트

## 1. 브랜드 개요
| 항목 | 내용 |

## 2. 제품 분석
- 제품 라인업
- 베스트셀러 TOP 5
- 가격 비교

## 3. SWOT 분석
| 강점 | 약점 |
|------|------|

| 기회 | 위협 |
|------|------|

## 4. 대응 전략
- 차별화 포인트
- 벤치마킹 포인트
- 액션 아이템
```
