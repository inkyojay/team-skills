# SmartStore Banner Creator

네이버 스마트스토어용 배너 템플릿 모음입니다.

## 사전 요구사항

```bash
pip install playwright
playwright install chromium
```

## 템플릿 종류

### 1. PC 메인배너 (1920×640)

스토어 상단 메인 배너입니다. 제품 이미지 나열 + 이벤트 정보.

**템플릿:** `pc-main-banner.html`

**변수:**
| 변수 | 설명 | 예시 |
|------|------|------|
| `{{EVENT_DATE}}` | 이벤트 기간 | 1/12 ~ 1/18 |
| `{{EVENT_TITLE}}` | 이벤트 타이틀 | KIDS WEEK |
| `{{EVENT_SUBTITLE}}` | 서브타이틀 | 최대 36% 할인 |
| `{{PRODUCT_IMAGE_1~5}}` | 제품 이미지 (5개) | ./images/product1.png |

**레이아웃:**
```
┌──────────────────────────────────────────────────────────┐
│  ●                                              ○       │
│    ◆        1/12 ~ 1/18                                 │
│            KIDS WEEK                           ◠        │
│            최대 36% 할인                                 │
│                                                          │
│   [제품1] [제품2] [제품3] [제품4] [제품5]      ●        │
│                                   ◡                      │
└──────────────────────────────────────────────────────────┘
```

### 3. PC 쿠폰배너 (1920×500)

PC용 쿠폰 안내 배너입니다.

**템플릿:** `coupon-banner-pc.html`

**변수:**
| 변수 | 설명 | 예시 |
|------|------|------|
| `{{MAIN_TITLE_1}}` | 메인 타이틀 1줄 | 매일 00시! |
| `{{MAIN_TITLE_2}}` | 메인 타이틀 2줄 | 선착순 쿠폰 3종 지급! |
| `{{SUB_TEXT}}` | 서브 텍스트 | 쿠폰은 내부 사정에 의해 조기 종료 될 수 있습니다. |
| `{{COUPON_VALUE_1}}` | 쿠폰1 값 | 1,000원 |
| `{{COUPON_VALUE_2}}` | 쿠폰2 값 | 10% |
| `{{COUPON_DESC_2}}` | 쿠폰2 설명 | 1만원 이상, 최대 3천원 할인 |
| `{{COUPON_VALUE_3}}` | 쿠폰3 값 | 10% |
| `{{COUPON_DESC_3}}` | 쿠폰3 설명 | 3만원 이상, 최대 5천원 할인 |

**레이아웃:**
```
┌──────────────────────────────────────────────────────────┐
│                                        ┌─────────────┐   │
│  매일 00시!                            │ COUPON      │   │
│  선착순 쿠폰 3종 지급!            ┌────│ 1,000원     │   │
│                                   │    └─────────────┘   │
│  쿠폰은 내부 사정에 의해...       │  ┌─────────────┐     │
│                                   └──│ COUPON 10%  │     │
│  ~~~~~~~~곡선 장식~~~~~~~~           │ 1만원 이상...│     │
│                                      └─────────────┘     │
│                                    ┌─────────────┐       │
│                                    │ COUPON 10%  │       │
│                                    │ 3만원 이상...│       │
│                                    └─────────────┘       │
└──────────────────────────────────────────────────────────┘
```

└── assets/

        ├── pc-main-banner.html # PC 메인배너 (1920×640)
        ├── mo-promo-banner.html # MO 프로모션 (750×1334)
