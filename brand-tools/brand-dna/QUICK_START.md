# 브랜드 DNA 추출 빠른 시작 가이드

## 설치 확인

```bash
# 필수 패키지
pip install playwright beautifulsoup4 requests

# Playwright 브라우저 설치
playwright install chromium
```

## 기본 사용법

### 1. 웹사이트 분석

```
이 사이트 브랜드 DNA 추출해줘
https://www.sundayhug.kr
```

### 2. 무드보드 생성

```
브랜드 무드보드 만들어줘
```

### 3. 경쟁사 비교

```
코니바이에린과 비교 분석해줘
```

## 분석 항목

| 항목 | 내용 |
|------|------|
| 컬러 팔레트 | 주요 색상, HEX 값 |
| 타이포그래피 | 폰트, 크기, 스타일 |
| 브랜드 퍼스낼리티 | 성격, 가치관 |
| 톤앤매너 | 커뮤니케이션 스타일 |
| 타겟 오디언스 | 핵심 고객층 |
| 비주얼 아이덴티티 | 이미지 스타일 |
| 포지셔닝 | 시장 내 위치 |

## 출력 위치

```
brand-tools/brand-dna/report/
├── brand-name/
│   ├── index.html        # 인터랙티브 리포트
│   ├── moodboard.png     # 무드보드
│   └── data.json         # 분석 데이터
```

## 분석 단계

1. 웹사이트 크롤링
2. 7단계 AI 분석
   - 퍼스낼리티
   - 톤앤매너
   - 타겟 오디언스
   - 비주얼 아이덴티티
   - 컬러 심리
   - 포지셔닝
   - 종합 분석
3. 리포트 생성

## 문제 해결

### 크롤링 실패
- JavaScript 렌더링 사이트는 대기 시간 증가
- 로봇 차단 사이트는 수동 분석 필요

### 이미지 추출 실패
- CDN 이미지는 정상 작동
- 로그인 필요 사이트는 접근 불가
