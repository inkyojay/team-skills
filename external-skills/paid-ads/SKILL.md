# 유료 광고 캠페인 스킬

Meta, Google, 네이버 등 유료 광고를 기획하고 최적화합니다.

## 플랫폼별 가이드

### Meta (Facebook/Instagram)

#### 캠페인 목표
| 목표 | 용도 | 최적화 기준 |
|------|------|------------|
| 인지도 | 브랜드 노출 | 도달, 노출 |
| 트래픽 | 웹사이트 방문 | 클릭 |
| 참여 | 인게이지먼트 | 좋아요, 댓글 |
| 리드 | 연락처 수집 | 리드 폼 |
| 판매 | 구매 전환 | 구매, ROAS |

#### 오디언스 설정
```markdown
## 핵심 타겟 (썬데이허그)
- 연령: 25-39세
- 성별: 여성
- 관심사: 육아, 임신, 출산준비
- 행동: 최근 쇼핑, 모바일 구매자

## 맞춤 타겟
- 웹사이트 방문자 (180일)
- 장바구니 이탈자 (30일)
- 구매자 (365일)
- 인스타그램 참여자

## 유사 타겟
- 구매자 유사 1%
- VIP 고객 유사 1%
- 높은 가치 고객 유사
```

#### 광고 형식
| 형식 | 사이즈 | 용도 |
|------|--------|------|
| 단일 이미지 | 1080×1080 | 제품 소개 |
| 캐러셀 | 1080×1080 × N | 다중 제품 |
| 영상 | 1080×1920 | 스토리/릴스 |
| 컬렉션 | 다양 | 카탈로그 |

#### 카피 가이드라인
```markdown
## Primary Text (125자)
- 첫 줄에 핵심 메시지
- 고객 고민 공감
- 혜택 강조

## 헤드라인 (40자)
- 행동 유도
- 숫자 활용
- 긴급성/희소성

## 설명 (30자)
- 추가 혜택
- CTA 보조
```

### Google Ads

#### 검색 광고
```markdown
## 키워드 유형
- 정확히 일치: [아기 베개]
- 구문 일치: "아기 베개"
- 확장 일치: 아기 베개

## 광고 문구
헤드라인 1: 프리미엄 아기 베개 | 썬데이허그
헤드라인 2: 역류 방지 5° 각도 설계
헤드라인 3: 지금 30% 할인 중

설명 1: 72,000명 엄마가 선택. 밤부 소재로 사계절 쾌적. 무료배송.
설명 2: 특허받은 인체공학 설계. 아기 숙면을 위한 최고의 선택.
```

#### 디스플레이 광고
```markdown
## 사이즈
- 반응형 (자동 조정)
- 300×250 (미디엄 직사각형)
- 336×280 (대형 직사각형)
- 728×90 (리더보드)
- 160×600 (와이드 스카이스크래퍼)
```

#### 쇼핑 광고
```markdown
## 필수 요소
- 고품질 제품 이미지
- 정확한 제목/설명
- 가격 업데이트
- 재고 상태
- 배송 정보
```

### 네이버 광고

#### 검색광고
```markdown
## 키워드 전략
- 브랜드: 썬데이허그
- 제품: 아기베개, 슬리핑백
- 일반: 신생아 용품, 출산 준비

## 광고 문구
제목: 썬데이허그 아기베개 | 역류방지 5° 설계
설명: 특허 기술로 안전한 수면. 밤부소재. 무료배송. 지금 할인 중!
```

#### 쇼핑검색광고
```markdown
## 최적화 포인트
- 상품명 키워드 포함
- 리뷰 수 & 평점
- 가격 경쟁력
- 빠른 배송
```

## 예산 배분

### 초기 테스트 (월 100만원)
```markdown
## 배분
- Meta: 60% (60만원)
  - 전환 캠페인: 40만원
  - 리타게팅: 20만원

- 네이버: 30% (30만원)
  - 브랜드 검색: 10만원
  - 쇼핑검색: 20만원

- 테스트: 10% (10만원)
  - 새로운 채널/소재
```

### 성장기 (월 500만원)
```markdown
## 배분
- Meta: 50% (250만원)
- Google: 20% (100만원)
- 네이버: 20% (100만원)
- 인플루언서: 10% (50만원)
```

## 성과 지표

### 핵심 KPI
| 지표 | 목표 | 계산 |
|------|------|------|
| ROAS | 300%+ | 매출 ÷ 광고비 |
| CPA | 15,000원 이하 | 광고비 ÷ 구매수 |
| CTR | 1%+ | 클릭 ÷ 노출 |
| CVR | 2%+ | 구매 ÷ 클릭 |

### 퍼널 지표
```markdown
노출 → 클릭 → 장바구니 → 결제 시작 → 구매

예시:
100,000 → 2,000 (2%) → 200 (10%) → 100 (50%) → 50 (50%)
```
