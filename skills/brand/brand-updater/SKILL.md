# 브랜드 업데이트

브랜드 정보를 업데이트하고 동기화합니다.

## 트리거
- "브랜드 정보 수정해줘"
- "제품 추가해줘"
- "경쟁사 업데이트"

## 업데이트 항목

### 1. 브랜드 가이드
- 톤앤매너 수정
- 컬러 팔레트 변경
- 권장/금지 표현 추가

### 2. 제품 정보
- 신제품 추가
- 가격 업데이트
- 스펙 수정

### 3. 경쟁사 정보
- 새 경쟁사 추가
- 분석 내용 업데이트

### 4. 캠페인 히스토리
- 완료된 캠페인 기록
- 성과 데이터 추가

## 사용법

```
"꿀잠슬리핑백 가격 59,000원으로 업데이트해줘"
"경쟁사에 코니바이에린 추가해줘"
"톤앤매너에 '전문가다운' 키워드 추가"
```

