# Meta 광고 카피 템플릿

## 5가지 카피 톤

### 1. 감성적 (Emotional)

감정에 호소하고 스토리텔링으로 공감대를 형성합니다.

**헤드라인 패턴:**
- "당신의 [고민]을 이해합니다"
- "[긍정적 결과]를 경험하세요"
- "매일 밤, [타겟]의 이야기"
- "[감정]을 선물하세요"

**Primary Text 패턴:**
```
[공감할 수 있는 상황 묘사]

[제품]이 [감정적 해결책]을 드립니다.

[따뜻한 마무리]
```

**예시 (육아용품):**
- 헤드라인: "아기가 5시간 연속 잠드는 비결"
- Primary: "매일 밤 아기와 함께 지새우는 시간, 저도 그랬습니다. 꿀잠슬리핑백과 함께 온 가족이 푹 잘 수 있었어요."

### 2. 기능적 (Functional)

제품의 기능, 스펙, 차별점을 명확하게 전달합니다.

**헤드라인 패턴:**
- "[핵심 기능]으로 [결과]"
- "[수치] [단위] [효과]"
- "[기술/소재]로 만든 [제품]"
- "NEW: [새로운 기능]"

**Primary Text 패턴:**
```
[제품명]의 핵심 기능:

✓ [기능 1]: [효과]
✓ [기능 2]: [효과]
✓ [기능 3]: [효과]

[스펙/인증 정보]
```

**예시 (육아용품):**
- 헤드라인: "모로반사 차단으로 깊은 수면"
- Primary: "꿀잠슬리핑백 핵심 기능:\n✓ 특허 모로반사 차단: 깜짝 놀람 방지\n✓ 4계절 온도조절: 36.5° 유지\n✓ OEKO-TEX 인증: 아기 피부에 안전"

### 3. 긴급성 (Urgency)

한정 기간, 재고 소진, 특별 할인으로 즉각적인 행동을 유도합니다.

**헤드라인 패턴:**
- "오늘만 [할인율]% OFF"
- "[N]시간 후 마감"
- "단 [N]개 남았습니다"
- "이번 주 한정 [혜택]"

**Primary Text 패턴:**
```
🔥 [프로모션명] 진행 중!

[할인/혜택 상세]

⏰ [마감 정보]
🎁 [추가 혜택]

놓치지 마세요!
```

**예시 (프로모션):**
- 헤드라인: "오늘만 30% + 무료배송"
- Primary: "🔥 블랙프라이데이 특가!\n\n정가 89,000원 → 62,300원\n\n⏰ 11월 30일 자정 마감\n🎁 선착순 100명 사은품 증정\n\n놓치지 마세요!"

### 4. 사회적 증거 (Social Proof)

구매후기, 판매량, 인증, 추천으로 신뢰를 구축합니다.

**헤드라인 패턴:**
- "[N]만 명이 선택한 [제품]"
- "★★★★★ [리뷰 수]개 후기"
- "[유명인/전문가] 추천"
- "[미디어] 선정 [수상]"

**Primary Text 패턴:**
```
"[실제 리뷰 인용]"
- [리뷰어 정보]

[판매량/평점 정보]

[추가 사회적 증거]
```

**예시 (육아용품):**
- 헤드라인: "72,000명이 선택한 수면 솔루션"
- Primary: "\"첫날부터 5시간 연속 잤어요!\" - 경기 김OO님\n\n★★★★★ 4.9점 (12,847개 리뷰)\n2024 대한민국 소비자대상 수상"

### 5. 문제-해결 (Problem-Solution)

고객의 고민을 정확히 짚고, 제품을 해결책으로 제시합니다.

**헤드라인 패턴:**
- "[고민], 이제 끝!"
- "[문제] 때문에 힘드셨나요?"
- "더 이상 [부정적 상황] 없이"
- "[해결책]으로 [긍정적 결과]"

**Primary Text 패턴:**
```
😫 이런 고민 있으셨나요?

• [문제 1]
• [문제 2]
• [문제 3]

✨ [제품]이 해결합니다!

[해결 방법 간단 설명]
```

**예시 (육아용품):**
- 헤드라인: "밤새 깨는 아기, 이제 끝!"
- Primary: "😫 이런 고민 있으셨나요?\n\n• 수시로 깨는 아기\n• 모로반사로 놀라 우는 아기\n• 잠들기까지 오래 걸리는 아기\n\n✨ 꿀잠슬리핑백이 해결합니다!"

## A/B 테스트 가이드

### 테스트 조합 예시

**헤드라인 테스트:**
| 버전 | 톤 | 헤드라인 |
|------|-----|----------|
| A | 감성 | 아기가 5시간 연속 잠드는 비결 |
| B | 문제-해결 | 밤새 깨는 아기, 이제 끝! |

**이미지 테스트:**
| 버전 | 프리셋 | 특징 |
|------|--------|------|
| A | product-hero | 제품 중심, 깔끔 |
| B | testimonial | 리뷰 강조, 신뢰 |

**CTA 테스트:**
| 버전 | CTA | 톤 |
|------|-----|-----|
| A | 지금 구매하기 | 직접적 |
| B | 더 알아보기 | 부드러움 |

### 성과 지표

| 테스트 대상 | 주요 지표 |
|-------------|----------|
| 헤드라인 | CTR (클릭률) |
| 이미지 | CTR, 체류시간 |
| CTA | 전환율 |
| 타겟 | ROAS, CPA |

## 업종별 카피 예시

### 육아용품

| 톤 | 헤드라인 |
|----|----------|
| 감성 | 아기의 첫 잠, 엄마의 첫 휴식 |
| 기능 | KC인증 완료, 안심하고 사용하세요 |
| 긴급성 | 출산 준비 SALE! 오늘까지 30% |
| 사회적 증거 | 10만 엄마가 인정한 필수템 |
| 문제-해결 | 수유 중 손목 통증, 해결됩니다 |
