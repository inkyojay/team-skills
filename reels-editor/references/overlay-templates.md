## FFmpeg 오버레이 예제

### 헤드라인 (반투명 배경)

```bash
ffmpeg -i input.mp4 \
  -vf "drawbox=x=0:y=298:w=iw:h=88:color=black@0.5:t=fill,\
       drawtext=fontfile='font.ttf':text='매일 밤 꿀잠을 선물하세요':\
       fontsize=48:fontcolor=white:x=(w-text_w)/2:y=318:\
       shadowcolor=black@0.8:shadowx=2:shadowy=2" \
  -c:v libx264 -c:a copy output.mp4
```

### CTA 버튼

```bash
ffmpeg -i input.mp4 \
  -vf "drawbox=x=290:y=1552:w=500:h=96:color=0x0095F6@0.9:t=fill,\
       drawtext=fontfile='font.ttf':text='지금 구매하기':\
       fontsize=36:fontcolor=white:x=(w-text_w)/2:y=1582" \
  -c:v libx264 -c:a copy output.mp4
```

### 단계 표시 (Step Indicator)

```bash
ffmpeg -i input.mp4 \
  -vf "drawbox=x=440:y=288:w=200:h=50:color=white@0.9:t=fill,\
       drawtext=fontfile='font.ttf':text='Step 1 / 3':\
       fontsize=28:fontcolor=black:x=(w-text_w)/2:y=300" \
  -c:v libx264 -c:a copy output.mp4
```

### 자막 스타일

```bash
ffmpeg -i input.mp4 \
  -vf "drawbox=x=50:y=1400:w=980:h=80:color=black@0.7:t=fill,\
       drawtext=fontfile='font.ttf':text='이렇게 하면 됩니다':\
       fontsize=32:fontcolor=white:x=(w-text_w)/2:y=1420" \
  -c:v libx264 -c:a copy output.mp4
```
