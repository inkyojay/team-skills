# 카드뉴스 구조 가이드

## 카드 유형별 용도

### 1. 커버 카드 (cover)
- **용도**: 시리즈 소개, 주제 제시, 관심 유도
- **구성**: 브랜드명, 에피소드 번호, 메인 타이틀, 서브타이틀
- **권장 위치**: 첫 번째 카드

### 2. 콘텐츠 카드 (content)
- **용도**: 본문 내용 전달
- **구성**: 브랜드명, 비주얼(이모지/이미지), 소제목, 본문(3-4문장)
- **권장 위치**: 2번째 ~ (마지막-2)번째
- **비주얼 이모지 예시**: 💡 (아이디어), 📌 (중요), ⚠️ (주의), ✨ (포인트), 🎯 (목표)

### 3. 인포 카드 (info)
- **용도**: 목록형 정보, 단계별 설명, 체크리스트
- **구성**: 브랜드명, 섹션 제목, 번호 매겨진 항목들
- **권장**: 3-5개 항목

### 4. 요약 카드 (summary)
- **용도**: 핵심 내용 정리
- **구성**: 브랜드명, 제목, 체크포인트 목록
- **권장 위치**: 마지막에서 두 번째

### 5. CTA 카드 (cta)
- **용도**: 행동 유도, 마무리
- **구성**: 브랜드명, 메시지, 액션 안내
- **권장 위치**: 마지막 카드

## 8-10장 구성 예시

```
1. cover    - 주제 소개
2. content  - 배경/문제 제기
3. content  - 핵심 개념 1
4. content  - 핵심 개념 2
5. info     - 주요 포인트 정리
6. content  - 실제 사례/적용
7. content  - 추가 팁
8. summary  - 핵심 요약
9. cta      - 마무리 (선택)
```

## 텍스트 작성 가이드

### 분량
- 카드당 3-4문장 (약 100-150자)
- 한 문장은 20-40자 권장

### 톤앤매너
- 친근하고 대화하는 듯한 어투
- "~해요", "~거든요", "~인가요?" 스타일
- 전문 용어는 쉽게 풀어서 설명

### 줄바꿈
- 의미 단위로 줄바꿈
- 강조할 부분은 별도 줄로 분리

## 색상 테마

| 테마 | 적합한 주제 |
|------|------------|
| 기본 (베이지/코랄) | 육아, 라이프스타일, 따뜻한 주제 |
| theme-blue | 기술, 비즈니스, 신뢰 |
| theme-green | 건강, 환경, 성장 |
| theme-purple | 창의성, 교육, 고급스러운 주제 |
