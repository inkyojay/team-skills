# 이메일 시퀀스 스킬

효과적인 이메일 마케팅 시퀀스를 생성합니다.

## 시퀀스 유형

### 1. 웰컴 시퀀스 (신규 가입)

```
Day 0: 환영 + 브랜드 스토리
Day 2: 베스트셀러 제품 소개
Day 5: 고객 후기 + 첫 구매 할인
Day 7: 육아 팁 콘텐츠
Day 14: 리마인더 + 할인 만료 알림
```

### 2. 장바구니 이탈 시퀀스

```
+1시간: "뭔가 잊으신 것 같아요"
+24시간: "아직 기다리고 있어요" + 제품 혜택
+72시간: "마지막 기회" + 추가 혜택
```

### 3. 구매 후 시퀀스

```
즉시: 주문 확인
+배송완료: 사용 가이드
+7일: 만족도 조사 + 리뷰 요청
+30일: 관련 제품 추천
+60일: 재구매 유도
```

### 4. 재활성화 시퀀스 (휴면 고객)

```
+30일 미방문: "보고 싶었어요"
+60일: 특별 할인 제안
+90일: "마지막 인사" + 최대 혜택
```

## 이메일 구조

### 제목줄 공식
1. **호기심**: "아기가 5시간 연속 잔 비결은?"
2. **혜택**: "[이름]님을 위한 30% 할인"
3. **긴급성**: "오늘 자정까지만!"
4. **개인화**: "[이름]님, 새로운 소식이 있어요"
5. **숫자**: "3가지 수면 팁"

### 본문 구조
```
1. 개인화된 인사
2. 훅 (주목 끄는 첫 문장)
3. 문제 공감
4. 해결책 제시
5. 사회적 증거
6. CTA 버튼
7. PS (추가 혜택/긴급성)
```

### CTA 버튼 텍스트
- ✅ "지금 구매하기"
- ✅ "할인 받기"
- ✅ "무료로 시작하기"
- ❌ "여기 클릭"
- ❌ "더 보기"

## 썬데이허그 웰컴 시퀀스 예시

### Day 0: 환영 이메일
```
제목: [이름]님, 썬데이허그 가족이 되신 것을 환영해요 💛

안녕하세요, [이름]님!

일요일 아침의 포옹처럼 따뜻한 썬데이허그에
오신 것을 진심으로 환영합니다.

저희는 아기의 편안한 수면을 위해
프리미엄 밤부 소재와 특허받은 디자인으로
최고의 제품을 만들고 있어요.

첫 구매 시 사용하실 수 있는
15% 할인 코드를 드릴게요!

[WELCOME15]

[지금 쇼핑하기]

PS. 이 코드는 7일 후 만료됩니다 ⏰
```

### Day 2: 베스트셀러 소개
```
제목: 72,000명의 엄마가 선택한 제품은? 👶

[이름]님, 안녕하세요!

가장 많은 사랑을 받고 있는
베스트셀러 3가지를 소개해드릴게요.

1️⃣ 밤부 슬리핑백
   "모로반사 때문에 자주 깨던 아기가
   이제 5시간씩 자요!" - 김○○님

2️⃣ 5도 각도 베개
   역류 방지 + 두상 관리를 한번에

3️⃣ 사계절 블랭킷
   통기성 좋은 밤부 소재

[베스트셀러 보러가기]
```

| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| 오픈율 | 25%+ | 제목줄 A/B 테스트 |
