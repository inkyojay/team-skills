# 메타 광고 이미지 규격

## 사이즈

| 배치 | 사이즈 (px) | 비율 | 용도 |
|------|------------|------|------|
| 피드 정사각 | 1080 x 1080 | 1:1 | 인스타/페북 피드 |
| 피드 세로 | 1080 x 1350 | 4:5 | 피드 최대 노출 (권장) |
| 스토리/릴스 | 1080 x 1920 | 9:16 | 스토리, 릴스 |
| 가로 | 1200 x 628 | 1.91:1 | 페북 링크 광고 |

## 안전 영역

```
┌─────────────────────────────┐
│  상단 안전 여백: 80px        │
│  ┌─────────────────────┐    │
│  │                     │    │
│  │   콘텐츠 영역        │    │  좌우 여백: 60px
│  │                     │    │
│  └─────────────────────┘    │
│  하단 안전 여백: 120px       │
│  (CTA 버튼 영역 고려)       │
└─────────────────────────────┘
```

- 상단 80px: 프로필/브랜드명 겹침 방지
- 하단 120px: "더 알아보기" CTA 버튼 겹침 방지
- 좌우 60px: 카드 라운드 코너 대비

## 텍스트 가이드

| 요소 | 사이즈 | 굵기 | 용도 |
|------|--------|------|------|
| 메인 헤드라인 | 48-56px | Bold (700) | 핵심 메시지 |
| 서브 카피 | 26-32px | Light-Regular (300-400) | 부가 설명 |
| 브랜드/제품명 | 22-28px | Medium (500) | 브랜드 식별 |
| 가격 (할인) | 52-64px | Bold (700) | 할인가 강조 |
| 가격 (원가) | 28-32px | Regular (400) + 취소선 | 원가 표시 |
| 뱃지 텍스트 | 20-24px | Medium-Bold (500-700) | New, LIVE 등 |
| 리뷰 텍스트 | 22-26px | Regular (400) | 고객 후기 |

## 컬러 가이드

기본 팔레트 (베이비/육아 브랜드):
- 배경: `#f5f0eb` (웜 크림), `#f0f0f0` (쿨 그레이), `#faf8f5` (아이보리)
- 텍스트: `#2c2520` (다크 브라운), `#6b5d52` (미디엄 브라운), `#8c7b6e` (라이트 브라운)
- 액센트: `#e84d3d` (레드 - LIVE/긴급), `#f0864a` (오렌지 - 할인), `#e8a0b4` (핑크 - New)

## 텍스트 비율 규칙

메타 광고에서 이미지 내 텍스트가 전체 면적의 20% 이하일 때 도달률이 최적화됨.
- 텍스트는 간결하게 (헤드라인 2줄 이내)
- 큰 폰트 + 적은 글자 수가 효과적
- 로고/브랜드명은 작게 배치
