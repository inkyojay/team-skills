# 리뷰 관리 스킬

고객 리뷰를 수집, 관리, 활용하는 전략을 수립합니다.

## 리뷰 수집 전략

### 요청 타이밍
```markdown
## 최적 시점
1. 배송 완료 후 3-5일 (사용 경험 후)
2. 재구매 시
3. CS 문의 해결 후 (만족 시)
4. 특별 혜택 제공 시

## 피해야 할 시점
- 배송 직후 (사용 전)
- 불만 접수 중
- 반품/환불 요청 시
```

### 요청 채널
```markdown
## 이메일
제목: [이름]님, 사용해보신 소감이 궁금해요 💛

안녕하세요, [이름]님!

[제품명] 잘 사용하고 계신가요?
저희 제품에 대한 솔직한 후기를 들려주세요.

리뷰 작성 시 다음 구매 시 사용 가능한
5,000원 할인 쿠폰을 드려요!

[리뷰 작성하기]

## 카카오 알림톡
썬데이허그입니다 💛
[제품명] 사용 소감은 어떠세요?
리뷰 남기시면 5,000원 할인 쿠폰 드려요!
[바로가기]

## 앱 푸시
📝 리뷰 쓰고 5,000원 쿠폰 받기
→ 탭하면 리뷰 페이지로 이동
```

### 리뷰 인센티브

|----------|------|------|

## 리뷰 응답 가이드

### 긍정 리뷰 응답
```markdown
## 템플릿
[이름]님, 소중한 후기 감사합니다 💛

아기가 편안하게 주무신다니 정말 기뻐요!
앞으로도 [이름]님 가족의 행복한 밤을 응원할게요.

다음에도 좋은 모습으로 찾아뵙겠습니다 🙏

## 포인트
- 구체적인 내용 언급
- 진심어린 감사
- 브랜드 가치 연결
```

### 부정 리뷰 응답
```markdown
## 템플릿
[이름]님, 불편을 드려 진심으로 사과드립니다.

[구체적 문제] 관련하여 확인 후
빠르게 조치해드리겠습니다.

고객센터(1234-5678)로 연락 주시면
담당자가 직접 도움 드릴게요.

다시 한번 사과드리며,
더 좋은 서비스로 보답하겠습니다.

## 원칙
1. 24시간 내 응답
2. 공감 먼저
3. 해결책 제시
4. 개인 채널로 유도
5. 공개적으로 책임 인정
```

### 상황별 응답

#### 배송 불만
```
배송 지연으로 불편 드려 죄송합니다.
현재 물량 증가로 배송이 다소 지연되고 있어요.
빠른 배송을 위해 노력하겠습니다.
```

#### 제품 불량
```
제품에 문제가 있어 정말 죄송합니다.
고객센터로 연락 주시면 즉시 교환/환불 처리해드릴게요.
더 꼼꼼한 품질 관리로 보답하겠습니다.
```

#### 기대 불일치
```
기대에 못 미쳐 죄송합니다.
고객님 의견을 제품 개선에 반영하겠습니다.
사용 중 불편하시면 고객센터로 연락 주세요.
```

## 리뷰 분석

### 분석 항목
```markdown
## 정량 분석
- 평균 평점
- 평점 분포 (5점/4점/3점/2점/1점)
- 리뷰 수 추이
- 사진 리뷰 비율

## 정성 분석
- 자주 언급되는 긍정 키워드
- 자주 언급되는 부정 키워드
- 개선 요청 사항
- 경쟁사 언급
```

### 키워드 분석 예시
```markdown
## 긍정 키워드 (밤부 베개)
1. "푹 잤어요" (152회)
2. "역류 방지" (98회)
3. "부드러워요" (87회)
4. "디자인 예뻐요" (65회)
5. "선물용으로" (43회)

## 부정 키워드
1. "세탁 방법" (23회) → FAQ 보강 필요
2. "높이가" (18회) → 사이즈 안내 개선
3. "냄새" (12회) → 첫 세탁 안내
```

### 인사이트 도출
```markdown
## 제품 개선
- [자주 언급되는 불만] → 제품 개선
- [요청 사항] → 신제품 기획

## 마케팅 활용
- [자주 언급되는 장점] → 광고 카피
- [고객 표현] → 상세페이지 반영

## CS 개선
- [반복 문의] → FAQ 추가
- [사용법 관련] → 가이드 제작
```

```markdown
## SNS
리뷰 리그램
베스트 리뷰 소개
```

### UGC (User Generated Content)
```markdown
## 활용 방법
1. 인스타 멘션 모니터링
2. 해시태그 검색 (#썬데이허그)
3. 좋은 콘텐츠 DM으로 허락 요청
4. 리그램 또는 광고 소재로 활용

## 보상
- 제품 제공
- 할인 쿠폰
- SNS 태그 & 감사
```
