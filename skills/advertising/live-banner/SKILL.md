# Live Banner Creator

쇼핑라이브용 배너 템플릿 모음입니다.

## 사전 요구사항

```bash
pip install playwright
playwright install chromium
```

## 템플릿 종류

### 1. 혜택 배너 (750×250)

라이브 시청 중 혜택 안내용 가로형 배너입니다.

**템플릿:** `benefit-banner.html`

**변수:**
| 변수 | 설명 | 예시 |
|------|------|------|
| `{{BENEFIT_NUM}}` | 혜택 번호 | 1, 2, 3... |
| `{{PRODUCT_TITLE}}` | 제품명 | 침대 암막 커버 |
| `{{DESCRIPTION_LINE_1}}` | 설명 1줄 | 라이브 시청 중 구매 고객 전 대상 |
| `{{DESCRIPTION_LINE_2}}` | 설명 2줄 (강조) | 암막 커버 증정 |
| `{{PRODUCT_IMAGE}}` | 제품 이미지 경로 | ./images/product.png |

**레이아웃:**
```
┌─────────────────────────────────────────────────────┐
│ ┌──────────┐                              ┌───────┐ │
│ │ 혜택 N  │  제품명                      │       │ │
│ └──────────┘                              │ 제품  │ │
│                                           │ 이미지│ │
│ 설명 1줄                                 │       │ │
│ 설명 2줄 (강조)                          └───────┘ │
└─────────────────────────────────────────────────────┘
```

### 2. 라이브 메인 배너 (720×1280)

쇼핑라이브 메인 화면용 세로형 배너입니다.

**템플릿:** `live-main-banner.html`

**변수:**
| 변수 | 설명 |
|------|------|
| `{{BRAND_NAME}}` | 브랜드명 |
| `{{PLATFORM}}` | 플랫폼명 (네이버 쇼핑라이브 등) |
| `{{MAIN_TITLE}}` | 메인 타이틀 |
| `{{PRODUCT_IMAGE_1~3}}` | 제품 이미지 (3개) |
| `{{MODE_TITLE_1~3}}` | 모드/제품 타이틀 |
| `{{MODE_SUBTITLE_1~3}}` | 영문 서브타이틀 |
| `{{MODE_DESC_1~3}}` | 상세 설명 |

**레이아웃:**
```
┌────────────────────┐
│ BRAND      LIVE 🔴 │
│                    │
│    메인 타이틀     │
│                    │
│ ┌────┐ 모드 1      │
│ │    │ 설명...     │
│ └────┘             │
│ ┌────┐ 모드 2      │
│ │    │ 설명...     │
│ └────┘             │
│ ┌────┐ 모드 3      │
│ │    │ 설명...     │
│ └────┘             │
└────────────────────┘
```

### 3. 이벤트 배너 (720×900)

라이브 예고 및 프로모션용 이벤트 배너입니다.

**템플릿:** `event-banner.html`

**변수:**
| 변수 | 설명 | 예시 |
|------|------|------|
| `{{BRAND_NAME}}` | 브랜드명 | SUNDAY HUG |
| `{{PLATFORM}}` | 플랫폼 | 네이버 쇼핑라이브 |
| `{{LIVE_DATE}}` | 라이브 날짜 | 2025. 12.18(목) |
| `{{LIVE_TIME}}` | 라이브 시간 | AM 11:00 |
| `{{EVENT_SUBTITLE}}` | 이벤트 부제 | 다신 없을 이 가격 |
| `{{PRODUCT_NAME}}` | 제품명 | ABC 접이식 아기침대 |
| `{{PRICE_LABEL}}` | 가격 라벨 | 단 한 시간만 |
| `{{PRICE}}` | 가격 | 159,000 |
| `{{PREVIEW_TIME}}` | 미리보기 시간 | AM11:00 |
| `{{PREVIEW_TITLE}}` | 미리보기 타이틀 | ABC 접이식 아기침대 |
| `{{PRODUCT_IMAGE}}` | 제품 이미지 | ./images/product.png |
| `{{DECORATION_EMOJI}}` | 장식 이모지 | 🎁 |

### 4. 프로모션 배너 (720×720)

정사각형 프로모션/이벤트 배너입니다.

**변수:**

## 사용 예시

```bash
# 혜택 배너 렌더링
python3 scripts/render_banner.py \
  --template benefit-banner.html \
  --output ./output/benefit-1.png \
  --vars '{"BENEFIT_NUM": "1", "PRODUCT_TITLE": "침대 암막 커버", ...}'
```

## 파일 구조

```
live-banner/
├── SKILL.md
└── assets/
    └── templates/
        ├── styles.css              # 공통 스타일
        ├── benefit-banner.html     # 혜택 배너 (750×250)
        ├── live-main-banner.html   # 라이브 메인 (720×1280)
        ├── event-banner.html       # 이벤트 배너 (720×900)
        ├── promo-banner.html       # 프로모션 배너 (720×720)
        ├── benefit-badge-banner.html # 방송중 혜택 뱃지 (750×250)
        ├── live-only-price.html    # 라이브 단독가 (720×1280)
        ├── daily-deal-grid.html    # 하루특가 그리드 (720×1280)
        └── customer-review.html    # 고객 후기 (720×1280)
```
