# Instagram Reels 규격 가이드

## 기본 규격

| 항목 | 규격 | 비고 |
|------|------|------|
| **화면 비율** | 9:16 | 세로 영상 |
| **해상도** | 1080×1920 | Full HD 세로 |
| **최대 길이** | 90초 | 권장: 15-60초 |
| **최소 길이** | 3초 | - |
| **파일 형식** | MP4 | H.264 코덱 권장 |
| **최대 파일 크기** | 4GB | - |
| **프레임 레이트** | 30fps | 권장 |

## 안전 영역 (Safe Zone)

Instagram UI 요소로 인해 상하단에 콘텐츠가 가려질 수 있습니다.

```
┌─────────────────────────┐
│   상단 안전 영역 (14%)   │  ← 사용자 아이디, 팔로우 버튼
│  약 268px               │
├─────────────────────────┤
│                         │
│                         │
│    안전 콘텐츠 영역      │  ← 핵심 콘텐츠
│    (72%)                │
│    약 1384px            │
│                         │
│                         │
├─────────────────────────┤
│   하단 안전 영역 (14%)   │  ← 캡션, 음악 정보, 버튼
│  약 268px               │
└─────────────────────────┘
```

### 안전 영역 수치

- **상단**: 0-268px (14%)
  - 사용자 아이디
  - 팔로우 버튼
  - 더보기 메뉴

- **하단**: 1652-1920px (14%)
  - 캡션 텍스트
  - 음악 정보
  - 좋아요/댓글/공유 버튼

- **측면**: 좌우 약 50px 여유 권장

## 텍스트 가이드라인

### 폰트 크기 권장

| 요소 | 크기 | 용도 |
|------|------|------|
| 헤드라인 | 48-64px | 주요 메시지 |
| 서브헤드 | 36-48px | 부가 설명 |
| CTA 버튼 | 32-40px | 행동 유도 |
| 캡션 | 24-32px | 세부 정보 |

### 텍스트 배치

- 안전 영역 내에 배치
- 중앙 또는 좌측 정렬 권장
- 배경 대비 충분한 명도 차이
- 그림자 또는 배경 박스 사용

## 오디오 규격

| 항목 | 규격 |
|------|------|
| 코덱 | AAC |
| 샘플 레이트 | 44.1kHz 또는 48kHz |
| 비트레이트 | 128kbps 이상 |
| 채널 | 스테레오 권장 |

## 광고용 추가 고려사항

### 권장 길이

- **브랜드 인지도**: 6-15초
- **제품 소개**: 15-30초
- **튜토리얼/데모**: 30-60초
- **스토리텔링**: 60-90초

### 처음 3초의 중요성

- 시청자 주의 집중 구간
- 브랜드/제품 노출 권장
- 강렬한 비주얼 또는 훅

### CTA (Call-to-Action)

효과적인 CTA 문구:
- "지금 구매하기"
- "더 알아보기"
- "무료 체험하기"
- "한정 할인 중"
- "지금 신청하세요"

## FFmpeg 변환 명령어

### 9:16 블러 배경 변환

```bash
ffmpeg -i input.mp4 \
  -filter_complex "[0:v]scale=1080:1920:force_original_aspect_ratio=increase,crop=1080:1920,boxblur=50:5[bg];[0:v]scale=1080:-1[fg];[bg][fg]overlay=(W-w)/2:(H-h)/2" \
  -c:v libx264 -preset medium -crf 23 \
  -c:a aac -b:a 128k \
  -movflags +faststart \
  output.mp4
```

### 90초 트리밍

```bash
ffmpeg -i input.mp4 -t 90 -c copy output.mp4
```

### 텍스트 오버레이

```bash
ffmpeg -i input.mp4 \
  -vf "drawtext=fontfile=/path/to/font.ttf:text='헤드라인':fontsize=48:fontcolor=white:x=(w-text_w)/2:y=300" \
  -c:v libx264 -c:a copy \
  output.mp4
```
