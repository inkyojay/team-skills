# CSV 데이터 분석 스킬

CSV/Excel 데이터를 분석하고 인사이트를 추출합니다.

## 분석 워크플로우

### Step 1: 데이터 탐색
```python
# 기본 정보 확인
- 행/열 수
- 컬럼 목록
- 데이터 타입
- 결측값 확인
- 기초 통계량
```

### Step 2: 데이터 정제
```python
# 전처리
- 결측값 처리
- 중복 제거
- 이상치 탐지
- 날짜 형식 통일
- 카테고리 정리
```

### Step 3: 분석 수행
```python
# 분석 유형
- 기술 통계
- 트렌드 분석
- 비교 분석
- 상관 분석
- 예측 분석
```

### Step 4: 시각화
```python
# 차트 유형
- 시계열: 라인 차트
- 비교: 바 차트
- 구성: 파이 차트
- 분포: 히스토그램
- 상관: 산점도
```

## Python 분석 스크립트

```python
import pandas as pd
import matplotlib.pyplot as plt

def analyze_sales(csv_path):
    """매출 데이터 기본 분석"""
    df = pd.read_csv(csv_path)

    # 기본 통계
    summary = {
        '총 매출': df['매출'].sum(),
        '평균 주문액': df['매출'].mean(),
        '주문 건수': len(df),
        '최고 매출일': df.groupby('날짜')['매출'].sum().idxmax()
    }

    # 제품별 매출
    product_sales = df.groupby('제품명')['매출'].sum().sort_values(ascending=False)

    return summary, product_sales
```
