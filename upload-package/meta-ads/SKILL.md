# Meta 광고 스킬

Meta(Facebook/Instagram) 광고 카피와 크리에이티브를 생성합니다.

## 기능

1. **카피 생성**: 5가지 톤으로 광고 카피 작성
2. **프리셋 추천**: 캠페인 목표에 맞는 광고 형식 추천
3. **A/B 테스트**: 여러 버전의 광고 세트 제안

## 워크플로우

### Step 1: 입력 분석

| 입력 유형 | 처리 방식 |
|-----------|-----------|
| 제품 URL | 웹사이트에서 정보 참고 |
| 제품명/정보 직접 입력 | 정보 정리 후 진행 |
| 프로모션/할인 광고 | 인터뷰 모드로 상세 요구사항 파악 |

**프로모션 광고 인터뷰 질문:**
1. 어떤 제품/서비스인가요?
2. 할인율 또는 혜택은 무엇인가요?
3. 프로모션 기간은 언제까지인가요?
4. 타겟 고객은 누구인가요?
5. 원하는 분위기는? (긴급함/프리미엄/친근함)

### Step 2: 카피 생성

5가지 톤으로 카피 생성:

| 톤 | 특징 | 예시 |
|----|------|------|
| 감성적 | 감정에 호소, 스토리텔링 | "아기가 5시간 연속 잠드는 비결" |
| 기능적 | 제품 기능/스펙 강조 | "모로반사 차단으로 깊은 수면" |
| 긴급성 | 한정 기간, 재고 소진 | "오늘만 30% + 무료배송" |
| 사회적 증거 | 리뷰, 판매량, 인증 | "72,000명이 선택한 수면 솔루션" |
| 문제-해결 | 고객 고민 → 해결책 | "밤새 깨는 아기, 이제 끝!" |

### Step 3: 프리셋 추천

**단일 이미지 (1080×1080)**
| 프리셋 | 용도 |
|--------|------|
| product-hero | 제품 중심 히어로 |
| benefit-focus | 혜택 강조 |
| testimonial | 후기/리뷰 |
| problem-solution | 문제-해결 |
| urgency-cta | 긴급성/CTA |
| lifestyle | 라이프스타일 |

**캐러셀 (1080×1080 × N장)**
| 프리셋 | 용도 |
|--------|------|
| feature-cards | 기능별 카드 5장 |
| step-by-step | 단계별 설명 4장 |
| before-after | 비포-애프터 3장 |
| product-lineup | 제품 라인업 N장 |

**스토리/릴스 (1080×1920)**
| 프리셋 | 용도 |
|--------|------|
| vertical-hero | 세로 히어로 |
| swipe-up-cta | 스와이프 CTA |
| countdown | 카운트다운 |

### Step 4: 출력

#### 카피 출력 형식

```markdown
# [제품명] Meta 광고 카피

## 캠페인 정보
| 항목 | 내용 |
|------|------|
| 제품 | [제품명] |
| 목표 | [전환/인지도/트래픽] |
| 타겟 | [타겟 설명] |

## 헤드라인 (5개)
1. [감성] ...
2. [기능] ...
3. [사회적 증거] ...
4. [문제-해결] ...
5. [긴급성] ...

## Primary Text (3개)
### 버전 A: 감성
...

### 버전 B: 기능
...

### 버전 C: 사회적 증거
...

## CTA 옵션
- 지금 구매하기
- 더 알아보기
- 할인 받기

## A/B 테스트 추천
| 테스트 | 버전 A | 버전 B |
|--------|--------|--------|
| 헤드라인 | #1 감성 | #4 문제-해결 |
| CTA | 지금 구매하기 | 할인 받기 |
```

## 텍스트 제한 (Meta 정책)

| 요소 | 최대 글자 수 | 권장 |
|------|-------------|------|
| Primary Text | 125자 | 80자 |
| 헤드라인 | 40자 | 25자 |
| 설명 | 30자 | 20자 |

## 이미지 규격

| 형식 | 사이즈 | 비율 |
|------|--------|------|
| 피드 단일 | 1080×1080 | 1:1 |
| 피드 세로 | 1080×1350 | 4:5 |
| 캐러셀 | 1080×1080 | 1:1 |
| 스토리/릴스 | 1080×1920 | 9:16 |
| 링크 광고 | 1200×628 | 1.91:1 |

## 프리셋 추천 기준

| 캠페인 목표 | 1순위 | 2순위 |
|-------------|-------|-------|
| 신제품 런칭 | product-hero | feature-cards |
| 프로모션/할인 | urgency-cta | countdown |
| 브랜드 인지도 | lifestyle | vertical-hero |
| 리타겟팅 | testimonial | problem-solution |
| 기능 설명 | benefit-focus | step-by-step |

## 사용 예시

### 예시 1: 제품 광고
```
사용자: 꿀잠슬리핑백 메타 광고 카피 만들어줘
→ 5가지 톤 카피 + 프리셋 추천 + A/B 테스트 제안
```

### 예시 2: 프로모션 광고
```
사용자: 블랙프라이데이 30% 할인 광고 만들어줘
→ 인터뷰 → 긴급성 톤 카피 + urgency-cta 프리셋
```

### 예시 3: 캐러셀 광고
```
사용자: 슬리핑백 기능 설명하는 캐러셀 카피 만들어줘
→ feature-cards 구성 (5장) + 각 카드별 카피
```

## 카피 템플릿

### 헤드라인 패턴

**감성:**
- "당신의 [고민]을 이해합니다"
- "[긍정적 결과]를 경험하세요"

**기능:**
- "[핵심 기능]으로 [결과]"
- "[수치] [단위] [효과]"

**긴급성:**
- "오늘만 [할인율]% OFF"
- "단 [N]개 남았습니다"

**사회적 증거:**
- "[N]만 명이 선택한 [제품]"
- "★★★★★ [리뷰 수]개 후기"

**문제-해결:**
- "[고민], 이제 끝!"
- "[문제] 때문에 힘드셨나요?"

### Primary Text 패턴

**감성:**
```
[공감할 수 있는 상황 묘사]

[제품]이 [감정적 해결책]을 드립니다.

[따뜻한 마무리]
```

**기능:**
```
[제품명]의 핵심 기능:

✓ [기능 1]: [효과]
✓ [기능 2]: [효과]
✓ [기능 3]: [효과]

[스펙/인증 정보]
```

**긴급성:**
```
🔥 [프로모션명] 진행 중!

[할인/혜택 상세]

⏰ [마감 정보]
🎁 [추가 혜택]

놓치지 마세요!
```
