# 제품 분석

제품 정보를 분석하여 마케팅 포인트를 도출합니다.

## 트리거
- "제품 분석해줘"
- "마케팅 포인트 뽑아줘"
- "USP 도출해줘"

## 분석 항목

### 1. 기본 정보
- 제품명, 카테고리
- 가격, 사이즈
- 소재, 원산지

### 2. 기능적 특징
- 핵심 기능
- 차별화 포인트
- 기술/특허

### 3. 감성적 가치
- 사용 경험
- 라이프스타일 연결
- 감정적 베네핏

### 4. 타겟 분석
- 주 사용자
- 구매 결정자
- 사용 시나리오
