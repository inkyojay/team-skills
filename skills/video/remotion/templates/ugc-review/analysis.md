# UGC 리뷰형 광고 분석

## 개요

| 항목 | 값 |
|------|-----|
| 유형 | UGC (User Generated Content) 리뷰형 |
| 권장 길이 | 30~45초 |
| 톤 | 캐주얼, 자연스러운 후기 |
| 타겟 | 제품 구매를 고려 중인 소비자 |

## 씬 구조

| # | 씬 | 시간 | 내용 | 비주얼 |
|---|-----|------|------|--------|
| 1 | 셀카 훅 | 0~5초 | 카메라 앞에서 직접 말하는 훅 | 셀카/거울 앵글, 일상 배경 |
| 2 | 검색 화면 | 5~10초 | 네이버/쇼핑 검색 화면 | 스크린 녹화 또는 모킹 UI |
| 3 | 제품 시연 | 10~22초 | 실제 제품 사용 모습 | 핸드헬드 촬영, 클로즈업 |
| 4 | 기능 소개 | 22~33초 | 제품 특징 하나씩 설명 | 클로즈업 + 텍스트 오버레이 |
| 5 | CTA | 33~38초 | 구매 유도 / 링크 안내 | 셀카 + 제품 샷 |

## 타이밍 패턴

- **씬 전환**: 직접 컷 (hard cut) 위주, 자연스러운 UGC 느낌
- **씬당 평균 길이**: 5~8초 (훅은 짧게 3~5초)
- **전체 페이스**: 빠르지 않고 자연스러운 리듬

## 텍스트 스타일

| 항목 | 값 |
|------|-----|
| 폰트 | 굵은 고딕 (Pretendard Black 또는 유사) |
| 크기 | 48~64px |
| 색상 | 흰색 기본, 강조 키워드 노랑(#FFE53B) |
| 위치 | 하단 1/3 영역 (bottom: 250~350px) |
| 외곽선 | 검정 stroke 2px |
| 그림자 | 강한 드롭쉐도우 (가독성) |
| 애니메이션 | 페이드인 + 약간의 슬라이드업 |

## 나레이션 스타일

- **톤**: 캐주얼, 친구에게 말하듯
- **성별**: 여성 (주로)
- **속도**: 보통~약간 빠름
- **특징**: "~거든요", "~진짜", "~대박" 등 구어체

## 레이아웃 패턴

```
┌─────────────────┐
│                  │
│   셀카/제품      │  ← 전체 배경 비디오
│   비디오         │
│                  │
│                  │
│  ┌────────────┐  │
│  │ 자막 Line1 │  │  ← 하단 자막 영역
│  │ 자막 Line2 │  │
│  └────────────┘  │
│                  │
└─────────────────┘
```

- **고정 영역**: 하단 자막 (항상 표시)
- **변동 영역**: 전체 배경 비디오 (씬마다 변경)
- **오버레이**: 하단 그라데이션 (텍스트 가독성)

## 핵심 특징

1. **셀카 훅**: 첫 씬에서 카메라를 직접 보며 시청자 관심 끌기
2. **검색 화면**: 실제 검색 행동을 보여줘 신뢰감 형성
3. **캐주얼 톤**: 광고가 아닌 진짜 후기처럼 느껴지는 자연스러움
4. **강조 키워드**: 핵심 단어만 노랑색으로 하이라이트
5. **UGC 스타일**: 핸드헬드 촬영, 일상 배경, 자연광
