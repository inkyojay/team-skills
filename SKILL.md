## 스킬별 상세 정보

### 1. 상세페이지 빌더
**폴더:** `page-builder/`

**요청 예시:**
- "상세페이지 만들어줘"
- "OO 제품 페이지 작성해줘"
- "이 URL 분석해서 상세페이지 리뉴얼해줘"
- "썬데이허그 스타일로 상세페이지 만들어줘"

**기능:**
- 이커머스 상품 상세페이지 HTML 생성
- 브랜드별 컬러 시스템 적용
- 기승전결 스토리텔링 구조
- 31종 섹션 템플릿 제공

---

### 2. 카드뉴스 제작
**폴더:** `card-news/`

**요청 예시:**
- "카드뉴스 만들어줘"
- "이 유튜브 영상으로 카드뉴스 만들어줘"
- "이 블로그 글 카드뉴스로 변환해줘"

**기능:**
- YouTube, 웹페이지, PDF에서 콘텐츠 추출
- 8-10장 카드 구조 자동 설계
- 나노바나 API로 배경 이미지 생성
- Sunday Hug 스타일 템플릿

---

### 3. 릴스 영상 편집
**폴더:** `reels-editor/`

**요청 예시:**
- "이 영상 릴스로 만들어줘"
- "영상 세로로 변환하고 자막 넣어줘"
- "10초~40초 구간만 잘라서 릴스로"

**기능:**
- Instagram Reels 규격 (9:16, 1080x1920) 변환
- AI 영상 분석 (Gemini Vision)
- 블러 배경 세로 변환
- 텍스트 오버레이, CTA 버튼 추가

**필수 설치:** `ffmpeg`, `GEMINI_API_KEY`

---

### 4. 메타 광고 기획
**폴더:** `meta-ads/`

**요청 예시:**
- "메타 광고 기획해줘"
- "인스타 광고 캠페인 만들어줘"
- "프로모션 광고 전략 세워줘"

**기능:**
- 대화형 광고 캠페인 전략 수립
- 소구점 발굴 및 컨셉 제안
- 타겟 오디언스 설계
- 크리에이티브 가이드 제작
