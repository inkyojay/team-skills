# 카피라이팅 스킬

설득력 있는 마케팅 카피를 작성합니다.

## 카피라이팅 프레임워크

### 1. AIDA
- **Attention** (주목): 눈길을 끄는 헤드라인
- **Interest** (흥미): 고객의 문제/욕구 자극
- **Desire** (욕망): 해결책과 혜택 제시
- **Action** (행동): 명확한 CTA

### 2. PAS
- **Problem** (문제): 고객의 고통점 명시
- **Agitate** (자극): 문제의 심각성 강조
- **Solution** (해결): 제품/서비스로 해결

### 3. BAB
- **Before** (전): 현재의 불편한 상태
- **After** (후): 제품 사용 후 이상적 상태
- **Bridge** (다리): 제품이 어떻게 변화시키는지

### 4. 4U 헤드라인
- **Urgent** (긴급): 지금 행동해야 하는 이유
- **Unique** (고유): 차별화된 포인트
- **Ultra-specific** (구체적): 숫자, 기간 명시
- **Useful** (유용): 고객에게 주는 혜택

## 카피 톤 가이드

### 감성적 (Emotional)
```
아기의 첫 꿀잠을 선물하세요 💤
매일 밤, 엄마의 품처럼 따뜻하게
```

### 기능적 (Functional)
```
특허받은 5° 각도 설계
모로반사 차단으로 5시간 연속 수면
```

### 긴급성 (Urgency)
```
오늘만! 연중 최저가 30% 할인
⏰ 남은 시간 23:59:59
```

### 사회적 증거 (Social Proof)
```
72,000명의 엄마가 선택한 이유
⭐ 4.9점 (3,847개 리뷰)
```

### 문제-해결 (Problem-Solution)
```
밤새 깨는 아기 때문에 지치셨나요?
이제 아침까지 푹 잘 수 있어요
```

## 플랫폼별 가이드

### 인스타그램 캡션
- 첫 줄에 핵심 메시지
- 150자 이내 권장
- 이모지 3개 이내
- 해시태그 5-10개

### 페이스북 광고
- Primary Text: 125자 이내
- 헤드라인: 40자 이내
- 설명: 30자 이내

### 네이버 스마트스토어
- 제목: 핵심 키워드 포함
- 상세 설명: 혜택 중심
- 리뷰 유도 문구

### 이메일 제목
- 10자 이내 핵심
- 호기심 유발
- 이모지 1개
- 개인화 ([이름]님)

## 썬데이허그 톤앤매너

### 핵심 키워드
- 따뜻함, 포근함, 안심
- 프리미엄, 전문성
- 밤부 소재, 친환경
- 숙면, 성장

### 피해야 할 표현
- 과장된 효과 주장
- 경쟁사 비방
- 불안 조장

### 권장 표현
- "일요일 아침의 포옹처럼"
- "아기와 함께하는 편안한 밤"
- "자연이 선물한 밤부 소재"

## 출력 형식

```markdown
# [제품/캠페인명] 카피 제안

## 헤드라인 (5개)
1. [감성] ...
2. [기능] ...
3. [긴급성] ...
4. [사회적 증거] ...
5. [문제-해결] ...

## 본문 카피

...

### 버전 B (기능)
...

## 플랫폼별 최적화
### 인스타그램
...

### 페이스북 광고
...
```
