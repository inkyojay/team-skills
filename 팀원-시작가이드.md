# AI 마케팅 스킬팩 - 팀원 시작 가이드

## 최초 설정 (딱 한 번만)

### 1. Claude Code 설치

터미널 열고 복붙:

```bash
npm install -g @anthropic-ai/claude-code
```

### 2. Claude 로그인

```bash
claude auth login
```

브라우저가 열리면 Claude 계정으로 로그인하세요.

---

## 매일 사용법

### 방법 1: 터미널에서 직접

```bash
cd /Users/inkyo/Projects/team-skills
claude
```

### 방법 2: 스크립트 실행 (더 간단)

```bash
./start.sh
```

또는 Finder에서 `start.sh` 더블클릭

---

## 사용 예시

Claude가 실행되면 자연어로 요청하세요:

| 요청 예시 | 설명 |
|----------|------|
| `카톡 배너 만들어줘` | 카카오 비즈메시지 배너 제작 |
| `상세페이지 작성해줘` | 제품 상세페이지 HTML 생성 |
| `브랜드 분석해줘` | 브랜드 DNA 추출 및 리포트 |
| `경쟁사 조사해줘` | 경쟁사 분석 리포트 |
| `릴스 편집해줘` | 인스타 릴스 영상 편집 |
| `메타 광고 만들어줘` | 인스타/페북 광고 소재 제작 |

## 문제 해결

### "command not found: claude"
→ Node.js 설치 필요: https://nodejs.org

### "인증 실패"
→ 다시 로그인: `claude auth login`

### "스킬을 찾을 수 없음"
→ 올바른 폴더에서 실행했는지 확인:
```bash
cd /Users/inkyo/Projects/team-skills
claude
```

## 결과물 저장 위치

모든 결과물은 `output/` 폴더에 저장됩니다:

```
output/
├── 광고카피/     # 광고 카피 문서
├── 영상/         # 영상 소재
├── 리포트/       # 분석 리포트
├── 상세페이지/   # 상세페이지 HTML
├── 카드뉴스/     # 카드뉴스 이미지
└── 기타/         # 기타 결과물
```
