# 썬데이허그 마케팅 스킬팩 사용 가이드

마케팅 업무 자동화를 위한 스킬 & 에이전트 패키지입니다.

---

## 사용 방법

**슬래시 명령어가 아닙니다.** 그냥 자연어로 요청하세요.

```
"상세페이지 만들어줘"
"카드뉴스 만들어줘"
"메타 광고 기획해줘"
```

Claude가 해당 스킬 파일을 참고해서 작업합니다.

---

## 전체 스킬 & 에이전트 목록

### 콘텐츠 제작 (4개)

| 스킬 | 요청 예시 | 기능 |
|------|----------|------|
| **상세페이지 빌더** | "상세페이지 만들어줘" | 이커머스 상품 상세페이지 HTML 생성 |
| **카드뉴스 제작** | "카드뉴스 만들어줘" | YouTube/웹/PDF에서 카드뉴스 자동 생성 |
| **릴스 영상 편집** | "릴스로 만들어줘" | Instagram Reels 규격 영상 편집 |
| **메타 광고 기획** | "광고 캠페인 기획해줘" | Meta 광고 전략 수립 및 크리에이티브 제작 |

## 스킬별 상세 정보

### 1. 상세페이지 빌더
**폴더:** `page-builder/`

**요청 예시:**
- "상세페이지 만들어줘"
- "OO 제품 페이지 작성해줘"
- "이 URL 분석해서 상세페이지 리뉴얼해줘"
- "썬데이허그 스타일로 상세페이지 만들어줘"

**기능:**
- 이커머스 상품 상세페이지 HTML 생성
- 브랜드별 컬러 시스템 적용
- 기승전결 스토리텔링 구조
- 31종 섹션 템플릿 제공

---

### 2. 카드뉴스 제작
**폴더:** `card-news/`

**요청 예시:**
- "카드뉴스 만들어줘"
- "이 유튜브 영상으로 카드뉴스 만들어줘"
- "이 블로그 글 카드뉴스로 변환해줘"

**기능:**
- YouTube, 웹페이지, PDF에서 콘텐츠 추출
- 8-10장 카드 구조 자동 설계
- 나노바나 API로 배경 이미지 생성
- Sunday Hug 스타일 템플릿

---

### 3. 릴스 영상 편집
**폴더:** `reels-editor/`

**요청 예시:**
- "이 영상 릴스로 만들어줘"
- "영상 세로로 변환하고 자막 넣어줘"
- "10초~40초 구간만 잘라서 릴스로"

**기능:**
- Instagram Reels 규격 (9:16, 1080x1920) 변환
- AI 영상 분석 (Gemini Vision)
- 블러 배경 세로 변환
- 텍스트 오버레이, CTA 버튼 추가

**필수 설치:** `ffmpeg`, `GEMINI_API_KEY`

---

### 4. 메타 광고 기획
**폴더:** `meta-ads/`

**요청 예시:**
- "메타 광고 기획해줘"
- "인스타 광고 캠페인 만들어줘"
- "프로모션 광고 전략 세워줘"

**기능:**
- 대화형 광고 캠페인 전략 수립
- 소구점 발굴 및 컨셉 제안
- 타겟 오디언스 설계
- 크리에이티브 가이드 제작

---

### 5. 브랜드 DNA 추출
**폴더:** `brand-tools/brand-dna/`

**요청 예시:**
- "브랜드 분석해줘"
- "이 사이트 DNA 추출해줘"
- "무드보드 만들어줘"

**분석 항목:**
- 컬러 팔레트
- 타이포그래피
- 브랜드 퍼스낼리티
- 톤앤매너

---

### 6. 제품 분석
**폴더:** `brand-tools/product-analyzer/`

**요청 예시:**
- "제품 분석해줘"
- "이 URL 제품 정보 정리해줘"

**기능:**
- 제품 URL에서 정보 수집
- 특징, 스펙, 가격 분석
- 분석 리포트 생성
