# 카드뉴스 생성

YouTube, 웹페이지, PDF를 인스타그램 카드뉴스로 자동 변환합니다.

## 트리거
- "카드뉴스 만들어줘"
- "인스타 콘텐츠 만들어줘"
- "이 영상으로 카드뉴스"

## 지원 소스
| 소스 | 설명 |
|------|------|
| YouTube | 자막 자동 추출 |
| 웹페이지 | 본문 콘텐츠 추출 |
| PDF | 문서 텍스트 추출 |
| 텍스트 | 직접 입력 |

## 카드 타입
| 타입 | 용도 |
|------|------|
| Cover | 메인 후크 (첫 장) |
| Content | 본문 + 이미지 |
| Full | 전체 이미지 배경 |
| Info | 리스트 형식 |
| Summary | 핵심 요약 |
| CTA | 행동 유도 (마지막 장) |

## 출력 규격
- 크기: 1080 x 1350px
- 포맷: PNG
- 장수: 5-10장

## 워크플로우
1. 소스 URL/텍스트 입력
2. 핵심 내용 추출 및 요약
3. 카드별 콘텐츠 구성
4. 디자인 적용
5. 이미지 파일 출력
