### 공통 코덱 설정

- 비디오: H.264 압축, 프로그레시브 스캔, 30fps 이하
- 오디오: AAC 스테레오, 128kbps 이상
- 픽셀: Square pixels, Fixed frame rate

## 3. 안전 영역 (Dead Zone)

### 릴스/스토리 (1080×1920)

```
┌──────────────────────┐
│   ⚠ 상단 250px       │  ← 프로필, 이름 영역
│   (텍스트/로고 금지)   │
├──────────────────────┤
│                      │
│   ✅ 안전 영역        │  1080 × 1420px
│   (핵심 콘텐츠)       │  (중앙)
│                      │
├──────────────────────┤
│   ⚠ 하단 250px       │  ← CTA 버튼, 캡션 영역
│   (텍스트/로고 금지)   │
└──────────────────────┘

좌우 여백: 각 6% (약 65px) 확보 권장
```

### 릴스 썸네일 노출 영역

```
릴스 9:16 (1080×1920) 업로드 시:

- 릴스 탭:       전체 9:16 노출
- 피드 탭:       중앙 4:5 (1080×1350) 영역만 노출
- 프로필 그리드:  중앙 1:1 (1080×1080) 영역만 노출

→ 핵심 요소는 반드시 중앙 1080×1080 영역 안에 배치
```

### 피드 이미지 (4:5)

```
┌──────────────────────┐
│← 좌우 75px 여백 권장 →│
│                      │
│   콘텐츠 영역         │  1080 × 1350px
│                      │
└──────────────────────┘
```

## 4. 텍스트 권장 길이

| 항목 | 권장 | 비고 |
|------|------|------|
| **기본 텍스트 (Primary)** | 125자 이내 | 초과 시 "더 보기" 처리 |
| **제목 (Headline)** | 40자 이내 | FB만 노출, IG 미사용 |
| **링크 설명** | 25자 이내 | FB만 노출 |
| **이미지 내 텍스트** | 20% 미만 | 강제 아님, 성과에 영향 |

---

## 5. 실전 제작 규격 (프로젝트 기준)

### 이미지 소재

| 소재 유형 | 해상도 | 용도 |
|----------|--------|------|
| 이미지 4:5 | 1080×1350px | 피드 최적 (기본) |
| 이미지 1:1 | 1080×1080px | 피드, 인스트림, 범용 |
| 이미지 9:16 | 1080×1920px | 스토리 전용 |

### 동영상 소재

| 소재 유형 | 해상도 | FPS | 용도 |
|----------|--------|-----|------|
| 영상 9:16 | 1080×1920px | 30fps | 릴스, 스토리 (기본) |
| 영상 4:5 | 1080×1350px | 30fps | 피드 동영상 최적 |
| 영상 1:1 | 1080×1080px | 30fps | 피드 범용 |

### Remotion 프로젝트 설정값

```typescript
// 릴스/스토리용 (9:16)
width: 1080,
height: 1920,
fps: 30,

// 피드용 (4:5)
width: 1080,
height: 1350,
fps: 30,

// 정사각형 (1:1)
width: 1080,
height: 1080,
fps: 30,
```

## 6. 제작 팁

- **하나로 커버**: 9:16 영상 하나로 릴스+스토리+피드 커버 가능 (피드 최적화는 4:5 별도 권장)
- **고해상도 업로드**: 1440×1440 이상 권장 (Meta 압축 후에도 품질 유지)
- **이미지 텍스트**: 20% 미만 유지 (강제 아니지만 도달/비용에 영향)
- **릴스 텍스트 제한**: 릴스 피드 이미지에는 텍스트 최소화 권장 (Meta 공식 가이드)
- **안전 영역 준수**: 9:16 제작 시 상하 250px에 핵심 요소 배치 금지
