# 비교 추천형 광고 분석

## 개요

| 항목 | 값 |
|------|-----|
| 유형 | 비교 추천형 (Comparison) |
| 권장 길이 | 45~60초 |
| 톤 | 정보 전달형, 객관적 비교 |
| 타겟 | 여러 제품을 비교 중인 구매 결정 단계 소비자 |

## 씬 구조

| # | 씬 | 시간 | 내용 | 비주얼 |
|---|-----|------|------|--------|
| 1 | 문제 제시 | 0~8초 | 소비자가 겪는 문제/고민 | 텍스트 중심 + 배경 영상 |
| 2 | 제품 비교 | 8~25초 | 여러 제품을 그리드로 비교 | 2~4칸 그리드 레이아웃 |
| 3 | 솔루션 시연 | 25~42초 | 우리 제품이 문제를 해결하는 모습 | 클로즈업 시연 영상 |
| 4 | 차별화 포인트 | 42~50초 | 핵심 차별점 강조 | 텍스트 + 아이콘 |
| 5 | CTA | 50~55초 | 구매 유도 | 제품 + 가격/링크 |

## 타이밍 패턴

- **씬 전환**: 슬라이드(wipe) 또는 페이드, 정보 전달형답게 부드러운 전환
- **씬당 평균 길이**: 8~15초 (비교 씬이 가장 긴 편)
- **전체 페이스**: 차분하지만 지루하지 않은 리듬

## 텍스트 스타일

| 항목 | 값 |
|------|-----|
| 폰트 | 둥근 고딕 (Pretendard 또는 유사) |
| 크기 | 42~56px |
| 색상 | 흰색 기본, 핵심 키워드 브랜드 컬러 |
| 스타일 | 둥근 필(pill) 배경 태그 |
| 위치 | 중앙 또는 하단 |
| 배경 | 반투명 둥근 배경 (pill shape) |
| 애니메이션 | 스케일인 + 페이드 |

## 나레이션 스타일

- **톤**: 전문적이지만 친근한 정보 전달
- **성별**: 남성 또는 여성
- **속도**: 보통
- **특징**: "~입니다", "~한데요" 등 설명체

## 레이아웃 패턴

### 비교 그리드 씬
```
┌─────────────────┐
│  "어떤 게 좋을까?"│  ← 상단 질문 텍스트
│                  │
│ ┌──────┬──────┐ │
│ │제품A │제품B │ │  ← 2x2 비교 그리드
│ ├──────┼──────┤ │
│ │제품C │제품D │ │
│ └──────┴──────┘ │
│                  │
│  [필 자막]       │  ← 하단 설명
└─────────────────┘
```

### 솔루션 씬
```
┌─────────────────┐
│                  │
│   시연 비디오     │  ← 전체 배경
│                  │
│                  │
│  ┌────────────┐  │
│  │ 필 자막    │  │  ← 둥근 배경 자막
│  └────────────┘  │
└─────────────────┘
```

## 핵심 특징

1. **문제 → 비교 → 솔루션 구조**: 논리적 설득 흐름
2. **그리드 비교**: 여러 제품을 한눈에 비교할 수 있는 레이아웃
3. **필 스타일 자막**: 둥근 배경의 태그형 자막으로 정보 전달
4. **객관적 톤**: 리뷰어가 아닌 정보 제공자 시점
5. **차별화 강조**: 비교를 통해 자연스럽게 우위 표현
