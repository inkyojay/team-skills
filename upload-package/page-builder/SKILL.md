# 상세페이지 빌더

이커머스 상품 상세페이지 HTML을 생성합니다.

## 트리거
- "상세페이지 만들어줘"
- "제품 페이지 작성"
- "[브랜드명] 스타일로"

## 워크플로우

### Step 1: 입력 수집
- URL 제공 시 → 크롤링 후 분석
- 제품 정보 직접 제공 → 바로 분석
- 분석 항목: 제품명, 주요 특징, 타겟 고객, 컬러 옵션, 가격, 사이즈 스펙

### Step 2: 구조 제안 (사용자 확인 필수)
반드시 사용자에게 섹션 구성을 확인받은 후 코드 작성:
1. 인트로 - 메인 카피
2. 후킹 멘트 - 스토리 방향
3. 포인트 01~03 - 각 특징
4. 사이즈 가이드
5. 브랜드 비전

### Step 3: HTML 코드 생성
- 인라인 CSS + JS 포함된 완전한 HTML 파일
- 더미 이미지: https://dummyimage.com/{width}x{height}/{bg}/{text}
- 브라우저에서 바로 열어 확인 가능

### Step 4: 최종 출력
- HTML 파일로 저장
- 이미지 교체 가이드 안내

## 브랜드 컬러
| 이름 | HEX | 용도 |
|------|-----|------|
| 다크 브라운 | #A38068 | 메인 포인트 |
| 데일리 크림 | #FFFBF5 | 밝은 배경 |
| 오트 | #EAE2D5 | 중간 배경 |
| 오렌지 | #ff8605 | CTA, 강조 |

## 금지 사항
- CSS 추가 작성 금지 (기존 클래스만 사용)
- 사용자 승인 없이 코드 작성 금지
