## 2. 영상 스펙 & 세이프존

### 필수 영상 규격

| 항목 | 규격 |
|------|------|
| 해상도 | **1080 x 1920px** |
| 비율 | **9:16** (세로) |
| 포맷 | MP4 / MOV (H.264 코덱) |
| 최대 용량 | 4GB |
| 오디오 | AAC 128kbps |
| 권장 길이 | **15~30초** (완료율 최적) |
| 최대 길이 | Instagram 15분, Facebook 무제한 |

### 세이프존 (1080x1920px 기준)

```
┌──────────────────────────────┐
│         상단 108px           │ ← 프로필, 릴스 표시
│  ┌──────────────────────┐    │
│  │                      │    │
│  │                      │ ←60px (좌) / 120px (우)
│  │    안전 영역           │    │
│  │    900 x 1492px      │    │
│  │    (핵심 콘텐츠 배치)   │    │
│  │                      │    │
│  └──────────────────────┘    │
│         하단 320px           │ ← CTA 버튼, 캡션, 좋아요/댓글
└──────────────────────────────┘
```

| 위치 | 여백 | 비율 |
|------|------|------|
| 상단 | 108px | ~14% |
| 하단 | 320px | ~35% |
| 좌측 | 60px | - |
| 우측 | 120px | - |
| **안전 영역** | **900 x 1492px** | 중앙 |

### 세이프존 주의사항

- **앱 인터페이스 표시 시**: 앱 주요 기능이 화면 아래에 배치되지 않아야 함
- **핵심 메시지 추가 시**: 제품 주요 기능을 중앙에 가깝게 배치 (효과 극대화)
- **CTA 적용 시**: 문구 CTA가 하단 UI에 가려지지 않아야 함
- **크리에이터 협력 시**: 안전 영역으로 인해 문구가 가려질 수 있음을 사전 안내

### 세이프존 확인 방법

- **모바일**: Instagram에서 `@thesafezonechecker` 검색 또는 모바일 AR 필터 사용
- **데스크톱**: Meta 제공 PPT / Keynote / PSD 템플릿 다운로드
- **광고 관리자**: Meta Ads Manager의 Safe Zone Guardrail 기능 활용

## 4. 스토리 유형 8가지

### 사진 콜라주
- 음악에 동기화되는 이미지/동영상 컬렉션
- 많은 노력 없이 제품의 실제 모습과 브랜드 개성 전달
- **관심 사로잡기**: 주요 컬렉션의 다양한 룩 소개
- **관심 유지하기**: 프로모션 혜택으로 타겟에게 어필
- **보상 제공하기**: 할인 가격 안내

### 유형 (Typecasting)
- 타겟이 공감할 수 있는 사람/물건/상황으로 표현
- 사람들이 콘텐츠에서 자기 자신을 볼 수 있음
- **관심 사로잡기**: 질문으로 타겟의 참여 유도
- **관심 유지하기**: 경쾌한 오디오 + 다양한 사람들
- **보상 제공하기**: 제목 카드로 브랜드 + 강력한 CTA

### 전환 시퀀스
- 적절한 음악과 함께 관심을 사로잡고 감정적 반응 유도
- 소품, 제품, 신체 움직임 등 모든 것을 활용
- **관심 사로잡기**: 발소리처럼 주의를 환기하는 오디오
- **관심 유지하기**: 비트에 동기화된 제품 전환 시퀀스
- **보상 제공하기**: 브랜드와 행동 유도가 담긴 제목 카드

### Q&A
- 두 사람의 대화로 감정을 유발하는 메시지 전달
- 새로운 타겟에게 브랜드 소개할 때 효과적
- **관심 사로잡기**: 관심 유도 요소로 호기심 자극
- **관심 유지하기**: 사운드 컬렉션의 오디오로 흥미 유발
- **보상 제공하기**: 질문에 대한 솔루션 = 제품

### 제품 데모
- 캐주얼하고 간단하게 제품 사용법 소개
- 의도에 따라 제품 사용 모습 클립 여러 개 연결
- **관심 사로잡기**: 기발한 동영상 촬영 기법
- **관심 유지하기**: 릴스 촬영 기법으로 주요 이점 소개
- **보상 제공하기**: 통일된 시각 언어로 매끄럽게 CTA 전환

### 비포 앤 애프터
- 제품 사용 전후의 감정적 변화를 담아냄
- "애프터"라는 단어와 이점을 텍스트에 추가
- **관심 사로잡기**: 타겟이 곧바로 알 수 있는 상황 설정 + 텍스트 스티커
- **관심 유지하기**: 보이스오버로 비포 앤 애프터 스토리라인
- **보상 제공하기**: 마지막 장면에서 "애프터" + 텍스트 스티커로 명확한 CTA

### 1인칭 시점
- 내레이터의 시점으로 타겟이 공감할 수 있는 상황 제시
- 누구나 경험한 상황을 보는 새로운 시각 제공
- **관심 사로잡기**: 1인칭 샷으로 시청자가 제품 사용 모습 상상
- **관심 유지하기**: 생생한 사운드 클립 추가
- **보상 제공하기**: 제품 경험을 깊게 즐기라고 권장

### 튜토리얼
- 복잡한 주제를 단계별로 나누어 즐기운 방식으로 전달
- 학습형 단계로 제품 강조 + 업계 관련 팁/모범 사례 공유
- **관심 사로잡기**: 처음 2초 안에 팁 소개
- **관심 유지하기**: 텍스트 스티커와 이미지/동영상으로 팁 2가지 + 이점 2가지
- **보상 제공하기**: 명확한 행동 유도와 제품 샷으로 마무리

## 6. 혼합 자산 캠페인 전략

### 왜 혼합해야 하는가

- 동일 광고 세트에 **정적(이미지) + 동적(비디오) 자산을 함께** 포함
- 이미지와 영상을 함께 사용하는 계정이 **50% 더 나은 성과**
- Meta Flexible Ad Format 활용 시 **25~40% 성과 개선**

### 운영 가이드

| 항목 | 권장 사항 |
|------|---------|
| 광고 세트당 변형 수 | 최대 3~4개 활성 변형 |
| 정적 이미지 수명 | 7~10일 |
| UGC 비디오 수명 | 14~18일 |
| 소재 교체 전략 | 수명 도래 전 사전 교체/리프레시 예약 |
| 소재 조합 | 정적(빠른 전달) + 동영상(제품 시연) 병행 |

### 정적 → 다이내믹 전환 팁

- 성과 좋은 이미지를 **그리드 레이아웃**으로 배치
- 하나의 릴스에서 **사진 콜라주**로 활용
- 음악에 동기화하여 다이내믹한 느낌 부여

## 7. 파트너십 광고

### 성과

- 클릭률 **13% 향상**
- CPA **19% 개선**

### 크리에이터 자격요건

- **Professional 계정** (Creator 또는 Business) 필수
- Meta 커뮤니티 가이드라인 및 브랜디드 콘텐츠 정책 준수
- 인증 배지(파란 체크) 보유 시 유리
- 진정성 있는 콘텐츠, 허위 주장 금지

### 파트너십 광고 코드 만드는 방법 (크리에이터 측)

1. **설정 > 계정 유형 > Professional 계정**으로 전환
2. **설정 > 크리에이터 도구 > 브랜디드 콘텐츠** > "브랜드 파트너 부스트 허용" 활성화
3. 콘텐츠 게시 시 **유료 파트너십 라벨**로 브랜드 태그
4. 게시물 `···` > **"파트너십 라벨 및 광고"** > "파트너십 광고 코드 받기" 활성화
5. 코드 복사 또는 DM으로 브랜드에 공유

### 브랜드 측 사용

- Meta Business Account + 광고 권한 필요
- 크리에이터가 공유한 코드를 Meta Ads Manager에 입력
- **크리에이터 마켓플레이스**: 브랜드가 적합한 크리에이터 검색/필터링/직접 소통
- **파트너십 광고 허브**: 권한 관리, 문제 해결, 콘텐츠 추천 확인
