# 🚀 Quick Start Guide

## 바로 시작하기

### 1. 프로젝트에 스킬 추가

Claude Desktop 또는 Claude.ai 프로젝트의 **Knowledge** 또는 **Skills** 폴더에 이 폴더를 업로드하세요.

### 2. 상세페이지 요청하기

**예시 프롬프트:**
```
슬리핑백 상세페이지 만들어줘.
- 제품명: 나바잠 슬리핑백
- 특징: 대나무 원단, 4계절 사용, 양방향 지퍼
- 컬러: 크림, 핑크, 그레이, 올리브, 브라운
- 가격: 45,000원
- 사이즈: S(0-6개월), M(6-18개월), L(18-36개월)
```

### 3. Claude가 하는 일

1. **분석**: 제품 정보 파악
2. **제안**: 섹션 구조 제안 → **사용자 확인**
3. **생성**: HTML 코드 작성
4. **제공**: 파일 다운로드 + 이미지 교체 가이드

## 🎨 더미 이미지 사용법

```
https://dummyimage.com/{너비}x{높이}/{배경색}/{텍스트색}
```

**자주 쓰는 조합:**
```
600x400/EAE2D5/A38068  - 메인 비주얼
600x600/FFFBF5/A38068  - 정사각형
100x100/컬러코드/000    - 컬러칩
```

## ⚠️ 주의사항

- ❌ 추가 CSS 작성 금지
- ❌ 외부 이미지 직접 링크 금지

## 💡 팁

3. **섹션 추가/삭제**: `section-templates.md`에서 복사해서 추가
