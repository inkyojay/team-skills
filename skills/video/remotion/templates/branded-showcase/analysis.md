# 브랜드 공식형 광고 분석

## 개요

| 항목 | 값 |
|------|-----|
| 유형 | 브랜드 공식형 (Branded Showcase) |
| 권장 길이 | 30~45초 |
| 톤 | 전문적, 공식적, 세련된 |
| 타겟 | 브랜드 인지도 형성, 신규 고객 |

## 씬 구조

| # | 씬 | 시간 | 내용 | 비주얼 |
|---|-----|------|------|--------|
| 1 | 브랜드 인트로 | 0~5초 | 브랜드 로고 + 제품명 | 고정 헤더 등장 + 배경 영상 |
| 2 | 특징 1 | 5~15초 | 핵심 특징 소개 | 헤더 고정 + 하단 비디오/이미지 |
| 3 | 특징 2 | 15~25초 | 두 번째 특징 | 헤더 고정 + 하단 비디오/이미지 |
| 4 | 특징 3 | 25~35초 | 세 번째 특징 | 헤더 고정 + 하단 비디오/이미지 |
| 5 | CTA | 35~40초 | 구매/방문 유도 | 헤더 + CTA 텍스트 |

## 타이밍 패턴

- **씬 전환**: 하단 영역만 슬라이드/페이드 전환, 헤더는 고정
- **씬당 평균 길이**: 8~10초
- **전체 페이스**: 안정적이고 일관된 리듬

## 텍스트 스타일

| 항목 | 값 |
|------|-----|
| 폰트 | 깔끔한 고딕 (Pretendard Medium~Bold) |
| 크기 | 40~52px (자막), 28~32px (헤더 텍스트) |
| 색상 | 흰색 기본, 핵심 키워드에 브랜드 컬러 하이라이트 |
| 위치 | 중앙~하단 (헤더 아래 영역) |
| 스타일 | 컬러 하이라이트 자막 (키워드에 색상 배경) |
| 애니메이션 | 페이드인 + 슬라이드업 |

## 나레이션 스타일

- **톤**: 전문적, 신뢰감 있는 브랜드 보이스
- **성별**: 남성 또는 여성 (브랜드 톤에 맞게)
- **속도**: 차분하고 여유있는 속도
- **특징**: "~합니다", "~드립니다" 등 격식체

## 레이아웃 패턴

```
┌─────────────────┐
│ ┌─────────────┐ │
│ │ 로고 제품명  │ │  ← 고정 상단 헤더 (항상 표시)
│ │ [제품 썸네일]│ │     로고 + 제품명 + 작은 제품 이미지
│ └─────────────┘ │
│                  │
│   비디오/이미지   │  ← 하단 콘텐츠 영역 (씬마다 변경)
│                  │
│  ┌────────────┐  │
│  │ 하이라이트  │  │  ← 컬러 하이라이트 자막
│  │   자막      │  │
│  └────────────┘  │
└─────────────────┘
```

- **고정 영역**: 상단 헤더 (로고 + 제품명 + 썸네일) - 전체 영상 동안 고정
- **변동 영역**: 하단 비디오/이미지 콘텐츠
- **오버레이**: 하단 그라데이션 + 컬러 하이라이트 자막

## 핵심 특징

1. **고정 헤더**: 브랜드 로고 + 제품명 + 썸네일이 항상 상단에 표시
2. **하단 콘텐츠 순환**: 헤더 아래 영역만 씬마다 변경
3. **컬러 하이라이트 자막**: 핵심 키워드에 브랜드 컬러 배경 하이라이트
4. **전문 나레이션**: 격식체로 브랜드 신뢰감 형성
5. **일관된 브랜딩**: 전체 영상에서 브랜드 아이덴티티 유지
