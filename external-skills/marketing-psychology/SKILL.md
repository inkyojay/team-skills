# 마케팅 심리학 스킬

구매 결정에 영향을 미치는 70+ 심리학 원칙을 활용합니다.

## 핵심 심리학 원칙

### 1. 사회적 증거 (Social Proof)
```markdown
## 적용 방법
- "72,000명의 엄마가 선택"
- 실시간 구매 알림
- 리뷰 수 강조
- 인플루언서 추천

## 썬데이허그 예시
"3,847개의 ⭐5점 리뷰"
"지금 15명이 보고 있어요"
```

### 2. 희소성 (Scarcity)
```markdown
## 적용 방법
- 한정 수량: "남은 재고 3개"
- 한정 시간: "오늘만 30% 할인"
- 독점 제품: "온라인 단독"

## 주의사항
- 거짓 희소성은 신뢰 손상
- 실제 재고 기반으로 표시
```

### 3. 긴급성 (Urgency)
```markdown
## 적용 방법
- 카운트다운 타이머
- 마감 시간 명시
- "오늘 주문 시 내일 도착"

## 예시
⏰ 세일 종료까지 23:59:59
```

### 4. 상호성 (Reciprocity)
```markdown
## 적용 방법
- 무료 샘플 제공
- 가치 있는 콘텐츠 (육아 가이드)
- 첫 구매 할인 쿠폰

## 썬데이허그 예시
"무료 수면 가이드 PDF 다운로드"
→ 이메일 수집 + 호감 형성
```

### 5. 권위 (Authority)
```markdown
## 적용 방법
- 전문가 추천
- 인증/수상 표시
- 특허 강조

## 예시
"소아과 의사 추천"
"KC 안전 인증"
"특허받은 5° 각도 설계"
```

### 6. 호감 (Liking)
```markdown
## 적용 방법
- 친근한 브랜드 톤
- 고객 스토리
- 비하인드 씬

## 썬데이허그 예시
"일요일 아침의 포옹처럼"
창업자 육아 스토리 공유
```

### 7. 일관성 (Commitment)
```markdown
## 적용 방법
- 작은 약속 → 큰 약속
- 무료 체험 → 유료 전환
- 뉴스레터 → 구매

## 예시
"무료 샘플 신청" → "정품 구매"
```

### 8. 앵커링 (Anchoring)
```markdown
## 적용 방법
- 원가 취소선 표시
- 프리미엄 옵션 먼저 제시
- 비교 기준점 설정

## 예시
~~89,000원~~ → 59,000원 (34% 할인)
```

### 9. 프레이밍 (Framing)
```markdown
## 적용 방법
- 긍정적 프레이밍: "95% 만족"
- 손실 회피: "놓치지 마세요"
- 이득 강조 vs 손실 강조

## 예시
❌ "5% 불만족"
✅ "95% 만족"
```

### 10. 디폴트 효과 (Default Effect)
```markdown
## 적용 방법
- 추천 옵션 미리 선택
- 인기 상품 표시
- 번들 기본 설정

## 예시
☑️ 베스트셀러 (기본 선택)
```

## 가격 심리학

### 9로 끝나는 가격
```
59,000원 vs 60,000원
→ 왼쪽 숫자 효과 (5만원대 인식)
```

### 가격 분할
```
"하루에 커피 한 잔 값 1,600원"
(월 49,000원)
```

### 미끼 효과 (Decoy)
```
S: 39,000원 (1개)
M: 59,000원 (2개) ← 미끼
L: 69,000원 (3개) ← 타겟
```

### 번들링
```
개별 구매: 150,000원
세트 구매: 99,000원 (34% 절약)
```

## 감정 트리거

| 감정 | 트리거 문구 | 적용 상황 |
|------|------------|----------|
| 두려움 | "아기 역류 위험" | 문제 제기 |
| 안심 | "안전하게 보호" | 해결책 제시 |
| 기쁨 | "첫 꿀잠의 행복" | 혜택 강조 |
| 죄책감 | "아기를 위한 최선" | 프리미엄 정당화 |
| 소속감 | "현명한 엄마들의 선택" | 사회적 증거 |
