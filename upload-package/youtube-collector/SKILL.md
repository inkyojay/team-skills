# 유튜브 수집

유튜브 채널을 등록하고 새 영상을 수집하여 자막 기반 요약을 생성합니다.

## 트리거
- "유튜브 채널 등록해줘"
- "새 영상 수집해줘"
- "영상 요약해줘"

## 기능

### 1. 채널 등록
- 채널 URL로 등록
- 관심 카테고리 지정
- 수집 주기 설정

### 2. 영상 수집
- 등록된 채널의 새 영상 자동 감지
- 자막(CC) 추출
- 메타데이터 수집

### 3. 요약 생성
- 자막 기반 핵심 내용 요약
- 타임스탬프별 주요 포인트
- 마케팅 인사이트 도출

## 데이터 저장
```
.reference/youtube/
├── channels.yaml     # 등록된 채널 목록
├── videos/           # 수집된 영상 데이터
│   └── {video_id}.yaml
└── summaries/        # 요약 문서
```

## 출력 형식
```
# [영상 제목] 요약

## 핵심 내용
1.
2.
3.

## 타임스탬프
- 00:00 -
- 05:30 -

## 마케팅 인사이트
-
```
