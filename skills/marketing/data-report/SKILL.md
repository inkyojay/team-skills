# 데이터 리포트 분석

마케팅 데이터를 분석하고 인사이트 리포트를 생성합니다.

## 트리거
- "데이터 분석해줘"
- "리포트 만들어줘"
- "성과 분석해줘"

## 분석 가능 데이터

### 판매 데이터
- 매출 추이
- 제품별 판매량
- 채널별 성과

### 마케팅 데이터
- 광고 성과 (CTR, CPC, ROAS)
- SNS 인게이지먼트
- 이메일 오픈율/클릭율

### 고객 데이터
- 유입 경로
- 구매 패턴
- 재구매율

## 출력 형식

```
# [기간] 성과 리포트

## 핵심 지표 (KPI)
| 지표 | 이번 달 | 전월 | 변화 |
|------|---------|------|------|
| 매출 | | | |
| 주문수 | | | |
| 객단가 | | | |

## 채널별 성과
### 자사몰
### 스마트스토어
### 쿠팡

## 인사이트
1.
2.
3.

## 액션 아이템
| 우선순위 | 액션 | 기대효과 |
|----------|------|----------|
```

## 시각화
- 차트/그래프 생성
- 트렌드 라인
- 비교 분석
