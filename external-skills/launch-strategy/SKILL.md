# 출시 전략 스킬

효과적인 제품/기능 출시 전략을 수립합니다.

## 출시 타임라인

### Phase 1: 사전 준비 (D-30~D-14)
```markdown
## 마케팅 자산 준비
- [ ] 제품 사진/영상 촬영
- [ ] 상세페이지 작성
- [ ] 광고 소재 제작
- [ ] 이메일 시퀀스 작성
- [ ] SNS 콘텐츠 준비

## 인프라 점검
- [ ] 재고 확보
- [ ] 배송 준비
- [ ] CS 스크립트 준비
- [ ] 웹사이트 테스트
```

### Phase 2: 티저 (D-14~D-7)
```markdown
## 기대감 조성
- [ ] 티저 이미지 공개
- [ ] 카운트다운 시작
- [ ] 인플루언서 시딩
- [ ] 얼리버드 신청 받기
- [ ] 뉴스레터 예고

## 콘텐츠 예시
"곧 공개됩니다... 🤫"
"D-7: 아기 수면의 새로운 기준"
```

### Phase 3: 프리론칭 (D-7~D-1)
```markdown
## 본격 홍보
- [ ] 제품 상세 공개
- [ ] 얼리버드 할인 오픈
- [ ] 인플루언서 리뷰 공개
- [ ] 언론 보도자료 배포
- [ ] 광고 캠페인 시작

## 긴급성 조성
"얼리버드 50명 한정"
"런칭 특가 D-3"
```

### Phase 4: 런칭 데이 (D-Day)
```markdown
## 활동
- [ ] 판매 오픈 공지
- [ ] 라이브 방송
- [ ] 실시간 SNS 업데이트
- [ ] 고객 문의 대응
- [ ] 주문 모니터링

## 콘텐츠
"드디어 오픈! 🎉"
"첫 100명 추가 혜택"
```

### Phase 5: 포스트 런칭 (D+1~D+14)
```markdown
## 모멘텀 유지
- [ ] 고객 후기 수집
- [ ] 리뷰 이벤트
- [ ] 추가 프로모션
- [ ] 성과 분석
- [ ] 다음 런칭 준비
```

## 채널별 전략

### 자사몰
```markdown
## 준비물
- 메인 배너
- 팝업 (얼리버드)
- 상세페이지
- FAQ

## 특별 혜택
- 첫 구매 할인
- 무료 배송
- 추가 증정품
```

### 스마트스토어
```markdown
## 준비물
- 상품 등록
- 키워드 최적화
- 리뷰 이벤트

## 런칭 전략
- 쿠폰 발급
- 라이브 커머스
```

### 인스타그램
```markdown
## 콘텐츠 플랜
D-7: 티저 릴스
D-5: 제품 스니크핏
D-3: 인플루언서 언박싱
D-1: 최종 카운트다운
D-Day: 런칭 공지 + 라이브
D+3: 고객 후기 리그램
```

### 이메일
```markdown
## 시퀀스
D-7: "새로운 소식이 있어요"
D-3: "곧 만나요" + 얼리버드 신청
D-1: "내일 오픈!" + 최종 리마인더
D-Day: "오픈했습니다!" + 구매 링크
D+3: "아직 고민 중이시라면" + 후기
```

## 인플루언서 시딩

### 선정 기준
```markdown
## 필수 조건
- 팔로워 1만~10만 (마이크로)
- 육아/라이프스타일 카테고리
- 인게이지먼트 3% 이상
- 진정성 있는 콘텐츠

## 우대 조건
- 신생아/영유아 자녀
- 이전 협업 경험
- 지역 영향력
```

### 시딩 패키지
```markdown
## 구성
- 제품 풀세트
- 브랜드 소개서
- 키 메시지 가이드
- 촬영 가이드라인
- 할인 코드 제공

## 요청사항
- 피드 1개 + 스토리 3개
- 솔직한 사용 후기
- 태그 & 해시태그
```

```markdown
- [ ] 제품 영상 1개+
- [ ] 인스타 릴스 3개
- [ ] 카드뉴스 1세트
- [ ] 이메일 템플릿 5개

- [ ] 재고 200개+
- [ ] CS FAQ 10개+

```
