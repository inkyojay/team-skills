# SEO 감사 스킬

웹사이트의 검색 엔진 최적화 상태를 분석합니다.

## 감사 체크리스트

### 1. 기술적 SEO

#### 크롤링 & 인덱싱
- [ ] robots.txt 확인
- [ ] sitemap.xml 존재 및 제출
- [ ] 404 에러 페이지 확인
- [ ] 리다이렉트 체인 없음
- [ ] 중복 콘텐츠 (canonical 태그)

#### 사이트 속도
- [ ] Core Web Vitals (LCP, FID, CLS)
- [ ] 이미지 최적화 (WebP, lazy loading)
- [ ] JS/CSS 최소화
- [ ] CDN 사용
- [ ] 캐싱 설정

#### 모바일
- [ ] 모바일 친화적 디자인
- [ ] 반응형 레이아웃
- [ ] 탭 타겟 크기
- [ ] 뷰포트 설정

### 2. 온페이지 SEO

#### 메타 태그
- [ ] 고유한 타이틀 (60자 이내)
- [ ] 메타 설명 (160자 이내)
- [ ] OG 태그 (소셜 공유)
- [ ] 파비콘

#### 콘텐츠
- [ ] H1 태그 (페이지당 1개)
- [ ] H2-H6 계층 구조
- [ ] 키워드 밀도 (2-3%)
- [ ] 내부 링크
- [ ] 이미지 alt 태그

#### URL 구조
- [ ] 짧고 설명적인 URL
- [ ] 키워드 포함
- [ ] 하이픈 사용 (언더스코어 X)
- [ ] HTTPS 적용

### 3. 오프페이지 SEO

#### 백링크
- [ ] 도메인 권위도
- [ ] 앵커 텍스트 다양성
- [ ] 독성 링크 제거
- [ ] 경쟁사 백링크 분석

#### 로컬 SEO
- [ ] 네이버 플레이스 등록
- [ ] 구글 마이비즈니스
- [ ] NAP 일관성 (이름, 주소, 전화)
- [ ] 지역 키워드

## 키워드 리서치 가이드

### 키워드 유형
1. **핵심 키워드**: 아기 베개, 신생아 베개
2. **롱테일**: 역류 방지 아기 베개 추천
3. **브랜드**: 썬데이허그 베개
4. **경쟁사**: 코니바이에린 대안

### 키워드 의도
- **정보성**: "아기 역류 원인" → 블로그 콘텐츠
- **상업성**: "아기 베개 추천" → 비교 페이지
- **거래성**: "아기 베개 구매" → 제품 페이지

## 구조화된 데이터 (Schema)

### 제품 스키마
```json
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "썬데이허그 밤부 베개",
  "description": "...",
  "brand": {
    "@type": "Brand",
    "name": "썬데이허그"
  },
  "offers": {
    "@type": "Offer",
    "price": "49000",
    "priceCurrency": "KRW"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.9",
    "reviewCount": "3847"
  }
}
```

### FAQ 스키마
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "밤부 베개는 언제부터 사용 가능한가요?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "신생아부터 사용 가능합니다..."
    }
  }]
}
```
