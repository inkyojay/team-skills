# 경쟁사 분석 스킬

경쟁사를 체계적으로 분석하고 전략적 인사이트를 도출합니다.

## 분석 프레임워크

### 1. 경쟁사 식별
```markdown
## 직접 경쟁사
동일 제품/서비스, 동일 타겟

## 간접 경쟁사
유사 제품/서비스, 다른 타겟

## 잠재 경쟁사
시장 진입 가능성이 있는 기업
```

### 2. 제품/서비스 분석
| 항목 | 우리 | 경쟁사 A | 경쟁사 B |
|------|------|----------|----------|
| 주력 제품 | | | |
| 가격대 | | | |
| 차별점 | | | |
| 약점 | | | |

### 3. 마케팅 분석
```markdown
## 메시지
- 태그라인
- 가치 제안
- 톤앤매너

## 채널
- 웹사이트
- 소셜미디어
- 광고 플랫폼

## 콘텐츠
- 블로그
- 영상
- 이메일
```

### 4. SWOT 분석
```markdown
## Strengths (강점)
-

## Weaknesses (약점)
-

## Opportunities (기회)
-

## Threats (위협)
-
```

## 썬데이허그 주요 경쟁사

### 코니바이에린
- **카테고리**: 아기띠, 슬리핑백
- **포지셔닝**: 프리미엄 미니멀
- **강점**: 브랜드 인지도, 디자인
- **약점**: 높은 가격대

### 스와들업
- **카테고리**: 속싸개, 슬리핑백
- **포지셔닝**: 기능성 중심
- **강점**: 모로반사 차단 특허
- **약점**: 디자인 다양성

### 에이든아네이
- **카테고리**: 모슬린 제품
- **포지셔닝**: 자연주의
- **강점**: 소재 품질
- **약점**: 가격

### 도노도노
- **카테고리**: 유아 침구
- **포지셔닝**: 가성비
- **강점**: 접근성
- **약점**: 프리미엄 이미지

## 경쟁 우위 확보 전략

### 1. 차별화
- 유니크한 가치 제안
- 특허/기술 우위
- 브랜드 스토리

### 2. 비용 리더십
- 효율적 운영
- 규모의 경제
- 가성비 포지셔닝

### 3. 집중화
- 니치 시장 공략
- 전문성 강화
- 고객 밀착
