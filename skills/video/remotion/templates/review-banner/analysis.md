# 리뷰 배너형 광고 분석

## 개요

| 항목 | 값 |
|------|-----|
| 유형 | 리뷰 배너형 (Review Banner) |
| 권장 길이 | 5~10초 |
| 톤 | 심플, 임팩트 있는 |
| 타겟 | 리타겟팅, 피드 광고, 빠른 노출 |

## 구성 요소

리뷰 배너형은 씬 기반이 아닌 **정적 컴포지트 레이아웃**입니다.

| 영역 | 위치 | 내용 |
|------|------|------|
| 아기 사진 | 상단 ~60% | 제품 사용 이미지/비디오 (배경) |
| 리뷰 카드 | 중앙 | 별점 + 리뷰 텍스트 말풍선 |
| 제품 배너 | 하단 ~30% | 제품 이미지 + 제품명 + 가격/CTA |

## 타이밍 패턴

- **전환**: 없음 (단일 컴포지트)
- **애니메이션**: 각 요소가 순차적으로 등장
  - 0~1초: 배경 페이드인
  - 1~2초: 리뷰 카드 스케일인
  - 2~3초: 제품 배너 슬라이드업
  - 3~6초: 정적 유지
- **전체 페이스**: 빠른 등장 → 정보 노출 유지

## 텍스트 스타일

| 항목 | 값 |
|------|-----|
| 리뷰 폰트 | 세리프 또는 손글씨풍 |
| 리뷰 크기 | 32~40px |
| 리뷰 색상 | 다크 그레이 (#333) |
| 제품명 폰트 | 굵은 고딕 |
| 제품명 크기 | 36~44px |
| 별점 색상 | 골드 (#FFD700) |

## 레이아웃 패턴

```
┌─────────────────┐
│                  │
│   아기 사진      │  ← 배경 이미지/비디오
│   (제품 사용)    │
│                  │
│ ┌──────────────┐ │
│ │ ★★★★★        │ │  ← 리뷰 카드 (말풍선)
│ │ "진짜 좋아요  │ │
│ │  강추합니다"  │ │
│ └──────────────┘ │
│                  │
│ ┌──────────────┐ │
│ │ [제품] 제품명 │ │  ← 제품 배너
│ │      가격/CTA │ │
│ └──────────────┘ │
└─────────────────┘
```

## 핵심 특징

1. **정적 컴포지트**: 씬 전환 없이 단일 레이아웃
2. **리뷰 카드**: 별점 + 실제 리뷰 텍스트로 소셜 프루프
3. **제품 배너**: 제품 이미지 + 이름 + 가격으로 즉각적 정보 전달
4. **짧은 길이**: 5~10초로 피드 스크롤 중 빠른 인식
5. **순차 등장**: 배경 → 리뷰 → 배너 순서로 시선 유도
