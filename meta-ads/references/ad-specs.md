# Meta 광고 규격 (2026)

## 이미지 규격

### 피드 광고

| 형식 | 사이즈 | 비율 | 파일 형식 | 최대 용량 |
|------|--------|------|-----------|-----------|
| 단일 이미지 (정사각) | 1080×1080 | 1:1 | JPG, PNG | 30MB |
| 단일 이미지 (세로) | 1080×1350 | 4:5 | JPG, PNG | 30MB |
| 링크 광고 | 1200×628 | 1.91:1 | JPG, PNG | 30MB |

### 캐러셀 광고

| 항목 | 규격 |
|------|------|
| 카드 수 | 2~10장 |
| 이미지 사이즈 | 1080×1080 (각 카드 동일) |
| 비율 | 1:1 |
| 파일 형식 | JPG, PNG |

### 스토리/릴스 광고

| 형식 | 사이즈 | 비율 | 파일 형식 |
|------|--------|------|-----------|
| 이미지 | 1080×1920 | 9:16 | JPG, PNG |
| 동영상 | 1080×1920 | 9:16 | MP4, MOV |

### 안전 영역 (Safe Zone)

스토리/릴스에서 텍스트와 중요 요소는 안전 영역 내에 배치:

```
┌─────────────────────┐
│     여백 (14%)      │  ← 상단 안전 영역
├─────────────────────┤
│                     │
│    안전 영역        │  ← 핵심 콘텐츠
│    (1080×1420)      │
│                     │
├─────────────────────┤
│     여백 (14%)      │  ← 하단 안전 영역 (CTA 버튼)
└─────────────────────┘
```

## 텍스트 제한

### 광고 카피

| 요소 | 최대 글자 수 | 권장 글자 수 |
|------|-------------|-------------|
| Primary Text | 125자 | 80자 |
| 헤드라인 | 40자 | 25자 |
| 설명 | 30자 | 20자 |
| 링크 설명 | 30자 | 20자 |

### 이미지 내 텍스트

- 이미지 면적의 20% 이하 권장
- 텍스트가 많으면 도달 감소 가능
- 로고, 워터마크는 예외

## 파일 요구사항

### 이미지

| 항목 | 요구사항 |
|------|----------|
| 형식 | JPG, PNG |
| 최대 용량 | 30MB |
| 해상도 | 최소 1080px (한 변) |
| 색상 모드 | sRGB |

### 동영상

| 항목 | 요구사항 |
|------|----------|
| 형식 | MP4, MOV |
| 최대 용량 | 4GB |
| 길이 | 1초~241분 (피드), 1~60초 (스토리) |
| 프레임률 | 30fps 이상 |
| 해상도 | 최소 1080×1080 |

## CTA 버튼 옵션

### 전환 목표

- 지금 쇼핑하기 (Shop Now)
- 지금 구매하기 (Buy Now)
- 지금 주문하기 (Order Now)
- 예약하기 (Book Now)
- 가입하기 (Sign Up)

### 트래픽/인지도 목표

- 더 알아보기 (Learn More)
- 자세히 보기 (See More)
- 웹사이트 방문 (Visit Website)
- 문의하기 (Contact Us)

### 앱 설치

- 앱 설치하기 (Install Now)
- 앱 사용하기 (Use App)
- 지금 플레이 (Play Now)

## 광고 형식별 권장사항

### 피드 단일 이미지

```
권장:
- 제품이 명확하게 보이는 고품질 이미지
- 간결한 텍스트 오버레이
- 브랜드 로고 포함 (작게)
- 대비가 높은 CTA 버튼

피해야 할 것:
- 과도한 텍스트
- 저해상도 이미지
- 복잡한 배경
- 여러 제품을 한 이미지에
```

### 캐러셀

```
권장:
- 첫 번째 카드에 가장 매력적인 이미지
- 일관된 디자인 언어
- 스토리텔링 구조
- 마지막 카드에 강력한 CTA

구성 예시:
Card 1: 메인 제품 히어로
Card 2: 기능/혜택 1
Card 3: 기능/혜택 2
Card 4: 사회적 증거 (리뷰)
Card 5: CTA + 가격
```

### 스토리/릴스

```
권장:
- 세로 화면 최적화
- 처음 3초 내 핵심 메시지
- 소리 없이도 이해 가능
- 자막/텍스트 오버레이

안전 영역:
- 상단 14%: 시스템 UI 영역
- 하단 14%: CTA 버튼 영역
- 핵심 콘텐츠는 중앙 72%에 배치
```

## 플랫폼별 차이

### Facebook

- 피드: 1:1 또는 4:5 권장
- 우측 컬럼: 1.91:1
- 마켓플레이스: 1:1

### Instagram

- 피드: 1:1 (정사각) 또는 4:5 (세로)
- 스토리: 9:16 필수
- 릴스: 9:16 필수
- 탐색: 1:1
