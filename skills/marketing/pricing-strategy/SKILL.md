# 가격 전략 스킬

효과적인 가격 책정과 수익화 전략을 수립합니다.

## 가격 책정 프레임워크

### 1. 원가 기반 (Cost-Plus)
```markdown
가격 = 원가 + 마진

예시:
- 제조원가: 15,000원
- 목표 마진: 60%
- 판매가: 15,000 ÷ 0.4 = 37,500원
```

### 2. 가치 기반 (Value-Based)
```markdown
고객이 인지하는 가치에 따른 가격

질문:
- 이 문제를 해결하면 얼마의 가치?
- 대안 대비 얼마나 더 좋은가?
- 고객이 지불 의향 있는 최대 금액?
```

### 3. 경쟁 기반 (Competition-Based)
```markdown
경쟁사 대비 포지셔닝

| 전략 | 가격 | 포지셔닝 |
|------|------|----------|
| 프리미엄 | +20~50% | 최고 품질 |
| 동등 | ±10% | 유사 가치 |
| 침투 | -20~30% | 시장 점유 |
```

## 심리적 가격 전략

### 매력적 가격 (Charm Pricing)
```
✅ 49,000원 (4만원대)
❌ 50,000원 (5만원대)

→ 왼쪽 숫자가 인식에 큰 영향
```

### 앵커링 (Anchoring)
```
~~89,000원~~ 59,000원

→ 원가 대비 할인 폭 강조
→ 항상 취소선 원가 먼저 표시
```

### 미끼 효과 (Decoy)
```
S: 39,000원 - 1개
M: 69,000원 - 2개 (미끼)
L: 79,000원 - 3개 ← 목표

→ M 옵션이 L을 더 매력적으로 만듦
```

### 가격 분할
```
"하루 1,600원" (월 49,000원)
"커피 한 잔 값으로"

→ 작은 단위로 부담감 감소
```

## 번들 전략

### 순수 번들
```markdown
세트로만 판매

예시:
- 신생아 필수 세트: 베개 + 블랭킷 + 슬리핑백
- 가격: 159,000원 (개별 219,000원)
- 할인율: 27%
```

### 혼합 번들
```markdown
개별 + 세트 모두 판매

예시:
- 베개: 59,000원
- 블랭킷: 79,000원
- 슬리핑백: 89,000원
- 세트: 189,000원 (16% 할인)
```

### 업셀 번들
```markdown
기본 제품 + 추가 옵션

예시:
- 베개: 59,000원
- 베개 + 커버: 69,000원 (+10,000원)
- 베개 + 커버 + 세탁망: 79,000원 (+20,000원)
```

## 프로모션 전략

### 할인 유형
| 유형 | 예시 | 적합 상황 |
|------|------|----------|
| % 할인 | 30% OFF | 고가 제품 |
| 정액 할인 | 10,000원 OFF | 저가 제품 |
| 1+1 | 하나 더 | 재고 처리 |
| 무료 배송 | 5만원 이상 | 객단가 상승 |
| 번들 할인 | 세트 20% | 교차 판매 |

### 프로모션 캘린더 (연간)
```markdown
## 1분기
- 1월: 새해 세일 (10-20%)
- 2월: 발렌타인 (커플 번들)
- 3월: 봄맞이 (환절기 제품)

## 2분기
- 4월: 봄 시즌
- 5월: 어린이날 (최대 세일)
- 6월: 여름 준비

## 3분기
- 7월: 여름 세일
- 8월: 백투스쿨
- 9월: 가을 시즌

## 4분기
- 10월: 가을 세일
- 11월: 블랙프라이데이 (연중 최대)
- 12월: 크리스마스, 연말 정산
```

## 가격 테스트 방법

### A/B 테스트
```markdown
## 테스트 설계
- 버전 A: 49,000원
- 버전 B: 59,000원
- 기간: 2주
- 지표: 전환율, 매출, 이익

## 주의사항
- 동일 제품, 동일 기간
- 충분한 트래픽 확보
- 외부 변수 통제
```

### 가격 민감도 조사
```markdown
## Van Westendorp 질문
1. 너무 싼 것 같은 가격?
2. 싸다고 느끼는 가격?
3. 비싸다고 느끼는 가격?
4. 너무 비싼 가격?

→ 최적 가격대 도출
```

